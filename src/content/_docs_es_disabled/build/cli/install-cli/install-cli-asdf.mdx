Si ya estás utilizando los gestores de paquetes ASDF o Mise, puedes instalar Aptos CLI con los siguientes pasos.

## Instalar con ASDF

<Steps>
  1. Verifica que tienes [ASDF](https://asdf-vm.com/) instalado.

  2. Instala el plugin de Aptos CLI.

     ```sh
     asdf plugin add aptos https://github.com/gregnazario/asdf-aptos.git
     ```

  3. Instala Aptos CLI.

     ```sh
     asdf install aptos latest
     ```

  4. (Opcional) Instala una versión específica.

     ```sh
     # Localmente
     asdf install aptos 7.9.0

     # Globalmente
     asdf global aptos 7.9.0
     ```

  5. Verifica que la instalación funcionó.

     ```sh
     aptos help
     ```

     Estas instrucciones de ayuda también sirven como una guía detallada útil para comandos específicos.
</Steps>

## Instalar con Mise

<Steps>
  1. Verifica que tienes [Mise](https://mise.jds.dev/) instalado.

  2. Instala el plugin de Aptos CLI.

     ```sh
     mise plugin add aptos https://github.com/gregnazario/asdf-aptos.git
     ```

  3. Instala Aptos CLI.

     ```sh
     mise install aptos
     ```

  4. (Opcional) Instala una versión específica.

     ```sh
     mise install aptos@7.9.0
     ```

  5. Añade la herramienta a tu configuración local.

     ```sh
     mise use aptos
     ```

  6. Verifica que la instalación funcionó.

     ```sh
     aptos help
     ```

     Estas instrucciones de ayuda también sirven como una guía detallada útil para comandos específicos.
</Steps>
