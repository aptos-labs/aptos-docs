هذا الدليل سيرشدك خلال إعداد بيئة التطوير للبناء على بلوك تشين أبتوس. سنغطي تثبيت الأدوات الأساسية وإعداد مساحة العمل الخاصة بك.

## المتطلبات الأساسية

قبل البدء، تأكد من أن لديك:

- **اتصال بالإنترنت** للتنزيلات والتحديثات
- **امتيازات المدير** على جهاز الكمبيوتر الخاص بك
- **محرر نصوص أو IDE** (نوصي بـ Visual Studio Code)

## 1. تثبيت واجهة سطر أوامر أبتوس

واجهة سطر الأوامر (CLI) هي الأداة الأساسية لتطوير أبتوس. اختر طريقة التثبيت حسب نظام التشغيل الخاص بك:

<Tabs>
  <TabItem label="macOS">
    <Steps>
      1. **باستخدام Homebrew (موصى به):**
         ```bash
         brew update
         brew install aptos
         ```

      2. **باستخدام سكريبت التثبيت:**
         ```bash
         curl -fsSL "https://aptos.dev/scripts/install_cli.py" | python3
         ```

      3. **التحقق من التثبيت:**
         ```bash
         aptos --version
         ```
    </Steps>
  </TabItem>

  <TabItem label="Linux">
    <Steps>
      1. **باستخدام سكريبت التثبيت:**
         ```bash
         curl -fsSL "https://aptos.dev/scripts/install_cli.py" | python3
         ```

      2. **إضافة إلى PATH (إذا لزم الأمر):**
         ```bash
         echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc
         source ~/.bashrc
         ```

      3. **التحقق من التثبيت:**
         ```bash
         aptos --version
         ```
    </Steps>
  </TabItem>

  <TabItem label="Windows">
    <Steps>
      1. **باستخدام PowerShell (تشغيل كمدير):**
         ```powershell
         iwr "https://aptos.dev/scripts/install_cli.py" -useb | py -
         ```

      2. **أو تنزيل الملف التنفيذي:**
         - اذهب إلى [صفحة الإصدارات](https://github.com/aptos-labs/aptos-core/releases)
         - قم بتنزيل `aptos-cli-x.x.x-Windows-x86_64.zip`
         - فك الضغط وأضف إلى PATH

      3. **التحقق من التثبيت:**
         ```cmd
         aptos --version
         ```
    </Steps>
  </TabItem>
</Tabs>

## 2. إعداد Node.js (للـ TypeScript SDK)

إذا كنت تخطط للاستخدام TypeScript SDK، ستحتاج إلى Node.js:

<Tabs>
  <TabItem label="macOS">
    ```bash
    # باستخدام Homebrew
    brew install node

    # أو باستخدام nvm (موصى به)
    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
    nvm install node
    nvm use node
    ```
  </TabItem>

  <TabItem label="Linux">
    ```bash
    # Ubuntu/Debian
    curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
    sudo apt-get install -y nodejs

    # أو باستخدام nvm (موصى به)
    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
    nvm install node
    nvm use node
    ```
  </TabItem>

  <TabItem label="Windows">
    1. قم بتنزيل Node.js من [nodejs.org](https://nodejs.org/)
    2. قم بتشغيل ملف التثبيت (.msi)
    3. اتبع معالج التثبيت

    ```cmd
    # التحقق من التثبيت
    node --version
    npm --version
    ```
  </TabItem>
</Tabs>

## 3. إعداد Python (للـ Python SDK)

إذا كنت تفضل Python:

<Tabs>
  <TabItem label="macOS">
    ```bash
    # Python عادة مثبت مسبقاً، ولكن للحصول على أحدث إصدار:
    brew install python3

    # تثبيت pip إذا لم يكن متاحاً
    python3 -m ensurepip --upgrade
    ```
  </TabItem>

  <TabItem label="Linux">
    ```bash
    # Ubuntu/Debian
    sudo apt update
    sudo apt install python3 python3-pip

    # CentOS/RHEL/Fedora
    sudo dnf install python3 python3-pip
    ```
  </TabItem>

  <TabItem label="Windows">
    1. قم بتنزيل Python من [python.org](https://python.org/)
    2. أثناء التثبيت، تأكد من تحديد "Add Python to PATH"
    3. التحقق:

    ```cmd
    python --version
    pip --version
    ```
  </TabItem>
</Tabs>

## 4. إعداد بيئة التطوير

### Visual Studio Code (موصى به)

<Steps>
  1. **تنزيل وتثبيت VS Code:**
     - اذهب إلى [code.visualstudio.com](https://code.visualstudio.com/)
     - قم بتنزيل الإصدار المناسب لنظام التشغيل الخاص بك

  2. **تثبيت الإضافات المفيدة:**
     ```text
     - Move Language (للعقود الذكية)
     - Rust (إذا كنت تخطط للمساهمة في أبتوس)
     - GitLens (لتحكم أفضل في Git)
     - REST Client (لاختبار APIs)
     ```

  3. **تكوين إعدادات المشروع:**
     أنشئ ملف `.vscode/settings.json` في مشروعك:
     ```json
     {
       "files.associations": {
         "*.move": "move"
       },
       "editor.tabSize": 2,
       "editor.insertSpaces": true
     }
     ```
</Steps>

## 5. إعداد مشروع جديد

الآن لننشئ مشروع تطوير أبتوس الأول:

<Tabs>
  <TabItem label="TypeScript">
    <Steps>
      1. **إنشاء دليل المشروع:**
         ```bash
         mkdir my-aptos-project
         cd my-aptos-project
         ```

      2. **تهيئة مشروع Node.js:**
         ```bash
         npm init -y
         ```

      3. **تثبيت مجموعة تطوير أبتوس:**
         ```bash
         npm install @aptos-labs/ts-sdk
         npm install -D typescript @types/node ts-node
         ```

      4. **إنشاء tsconfig.json:**
         ```json
         {
           "compilerOptions": {
             "target": "ES2020",
             "module": "commonjs",
             "lib": ["ES2020"],
             "outDir": "./dist",
             "rootDir": "./src",
             "strict": true,
             "esModuleInterop": true,
             "skipLibCheck": true,
             "forceConsistentCasingInFileNames": true,
             "resolveJsonModule": true
           },
           "include": ["src/**/*"],
           "exclude": ["node_modules", "dist"]
         }
         ```

      5. **إنشاء هيكل المشروع:**
         ```bash
         mkdir src
         touch src/index.ts
         ```
    </Steps>
  </TabItem>

  <TabItem label="Python">
    <Steps>
      1. **إنشاء دليل المشروع:**
         ```bash
         mkdir my-aptos-project
         cd my-aptos-project
         ```

      2. **إنشاء بيئة افتراضية:**
         ```bash
         python3 -m venv venv

         # تفعيل البيئة الافتراضية
         # على macOS/Linux:
         source venv/bin/activate
         # على Windows:
         # venv\Scripts\activate
         ```

      3. **تثبيت مجموعة تطوير أبتوس:**
         ```bash
         pip install aptos-sdk
         ```

      4. **إنشاء requirements.txt:**
         ```bash
         pip freeze > requirements.txt
         ```

      5. **إنشاء هيكل المشروع:**
         ```bash
         touch main.py
         mkdir src
         ```
    </Steps>
  </TabItem>

  <TabItem label="Move (العقود الذكية)">
    <Steps>
      1. **إنشاء مشروع Move:**
         ```bash
         aptos move init my-move-project
         cd my-move-project
         ```

      2. **استكشاف هيكل المشروع:**
         ```text
         my-move-project/
         ├── Move.toml          # ملف تكوين المشروع
         ├── sources/           # ملفات Move المصدرية
         ├── tests/            # اختبارات الوحدة
         └── scripts/          # سكريبتات النشر
         ```

      3. **فهم Move.toml:**
         ```toml
         [package]
         name = "my-move-project"
         version = "1.0.0"
         authors = ["Your Name <your.email@example.com>"]

         [addresses]
         MyAddress = "_"

         [dev-addresses]
         MyAddress = "0x1"

         [dependencies.AptosFramework]
         git = "https://github.com/aptos-labs/aptos-core.git"
         rev = "mainnet"
         subdir = "aptos-move/framework/aptos-framework"
         ```
    </Steps>
  </TabItem>
</Tabs>

## 6. إعداد محفظة التطوير

لاختبار التطبيقات، ستحتاج إلى محفظة:

<Steps>
  1. **إنشاء حساب جديد:**
     ```bash
     aptos init
     ```

  2. **اختيار الشبكة:**
     - `testnet` - للتطوير والاختبار
     - `devnet` - لأحدث الميزات (غير مستقر)
     - `mainnet` - للإنتاج (استخدم عملات حقيقية)

  3. **الحصول على عملات تجريبية:**
     ```bash
     aptos account fund-with-faucet --account default
     ```

  4. **التحقق من الرصيد:**
     ```bash
     aptos account balance --account default
     ```
</Steps>

## 7. اختبار الإعداد

لنتأكد من أن كل شيء يعمل بشكل صحيح:

<Tabs>
  <TabItem label="TypeScript">
    أنشئ ملف `src/test-setup.ts`:

    ```typescript
    import { Aptos, AptosConfig, Network, Account } from "@aptos-labs/ts-sdk";

    async function testSetup() {
      // إعداد عميل أبتوس
      const config = new AptosConfig({ network: Network.TESTNET });
      const aptos = new Aptos(config);

      // إنشاء حساب جديد
      const account = Account.generate();
      console.log("عنوان الحساب:", account.accountAddress.toString());

      // التحقق من اتصال الشبكة
      try {
        const chainId = await aptos.getChainId();
        console.log("معرف السلسلة:", chainId);
        console.log("✅ الإعداد ناجح!");
      } catch (error) {
        console.error("❌ خطأ في الاتصال:", error);
      }
    }

    testSetup();
    ```

    تشغيل الاختبار:

    ```bash
    npx ts-node src/test-setup.ts
    ```
  </TabItem>

  <TabItem label="Python">
    أنشئ ملف `test_setup.py`:

    ```python
    from aptos_sdk.client import RestClient
    from aptos_sdk.account import Account

    def test_setup():
        # إعداد عميل أبتوس
        client = RestClient("https://fullnode.testnet.aptoslabs.com/v1")
        
        # إنشاء حساب جديد
        account = Account.generate()
        print(f"عنوان الحساب: {account.address()}")
        
        # التحقق من اتصال الشبكة
        try:
            chain_id = client.chain_id
            print(f"معرف السلسلة: {chain_id}")
            print("✅ الإعداد ناجح!")
        except Exception as error:
            print(f"❌ خطأ في الاتصال: {error}")

    if __name__ == "__main__":
        test_setup()
    ```

    تشغيل الاختبار:

    ```bash
    python test_setup.py
    ```
  </TabItem>

  <TabItem label="Move">
    أنشئ عقد بسيط في `sources/hello_world.move`:

    ```move
    module MyAddress::HelloWorld {
        use std::debug;
        use std::string;

        fun hello_world() {
            debug::print(&string::utf8(b"Hello, Aptos World!"));
        }

        #[test]
        fun test_hello_world() {
            hello_world();
        }
    }
    ```

    تشغيل الاختبار:

    ```bash
    aptos move test
    ```
  </TabItem>
</Tabs>

## المشاكل الشائعة وحلولها

<Aside type="tip">
  ### مشكلة: "aptos: command not found"

  **الحل:** تأكد من إضافة مسار أبتوس إلى متغير البيئة PATH:

  ```bash
  echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc
  source ~/.bashrc
  ```
</Aside>

<Aside type="tip">
  ### مشكلة: خطأ في الاتصال بالشبكة

  **الحل:**

  1. تحقق من اتصال الإنترنت
  2. جرب شبكة مختلفة (testnet/devnet)
  3. تحقق من إعدادات جدار الحماية
</Aside>

<Aside type="tip">
  ### مشكلة: فشل تجميع Move

  **الحل:**

  1. تحقق من صحة بناء جملة Move
  2. تأكد من أن dependencies محدثة
  3. احذف مجلد `build` وأعد المحاولة
</Aside>

## الخطوات التالية

الآن بعد أن أصبحت بيئة التطوير جاهزة:

1. **جرب [معاملتك الأولى](/ar/build/guides/first-transaction)** - تعلم كيفية إرسال المعاملات

4. **اقرأ [وثائق مجموعة التطوير](/ar/build/sdks)** - تعمق في الوظائف المتاحة

<Aside type="note">
  تذكر أن تحتفظ بأدواتك محدثة بانتظام للحصول على أحدث الميزات والإصلاحات الأمنية:

  ```bash
  # تحديث أبتوس CLI
  aptos update

  # تحديث مجموعات التطوير
  npm update    # للـ TypeScript
  pip install --upgrade aptos-sdk    # للـ Python
  ```
</Aside>
