لمعرفة المزيد عن الاختلافات والتشابهات، راجع [Aptos Learn](https://learn.aptoslabs.com/en/tutorials/solana-to-aptos-guide/cheat-sheet?workshop=solana-to-aptos)

|                              | Solana                                                                                                             | Aptos                                                                 |
|------------------------------|--------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------|
| **العقود الذكية**           | Rust, SVM                                                                                                          | Move, MoveVM                                                          |
| **رسوم المعاملات**          | منخفضة                                                                                                             | منخفضة                                                                |
| **التوازي**                 | توازي تشاؤمي، تحتاج إلى الإعلان عن جميع حسابات الكتابة                                                           | توازي تفاؤلي، السلسلة تستنتج حسابات الكتابة نيابة عنك                 |
| **دعم حساب العقد**          | حساب PDA                                                                                                           | [كائن](/build/smart-contracts/objects) أو [حساب مورد](/build/smart-contracts/resource-accounts)(يُنصح باستخدام الكائن بدلاً من ذلك) |
| **تخزين البيانات**         | تخزين البيانات في حساب مملوك للبرامج                                                                             | تخزين البيانات كمورد في حساب المستخدم أو كائن                         |
| **مستوى التخزين**           | مستوى البرنامج                                                                                                    | عام عند تخزينه في كائن                                               |
| **عقلية التخزين**           | بيانات المستخدم مخزنة بشكل موزع في الحساب                                                                          | بيانات المستخدم مخزنة بشكل موزع في الكائن                             |
| **مثال الكود**              | [عقد قائمة المهام في Solana](https://github.com/aptos-labs/move-by-examples/tree/main/advanced-todo-list/solana) | [عقد قائمة المهام في Aptos](https://github.com/aptos-labs/move-by-examples/tree/main/advanced-todo-list/aptos) |
| **معرف المتصل**             | `signer`                                                                                                           | `signer`                                                              |
| **إمكانية الترقية**         | البرنامج قابل للترقية                                                                                             | الوحدة قابلة للترقية                                                 |
| **نوع الإرسال**             | إرسال ثابت                                                                                                        | إرسال ثابت                                                            |
| **معيار FT**                | برنامج الرمز المميز                                                                                               | [Coin](/build/smart-contracts/aptos-coin) (legacy) و [معيار الأصل القابل للاستبدال](/build/smart-contracts/fungible-asset) |
| **معيار NFT**               | برنامج الرمز المميز                                                                                               | [معيار الأصل الرقمي](/build/smart-contracts/digital-asset)             |
| **تفاعل البلوكشين**         | مكتبة Solana web3.js                                                                                              | [Aptos TypeScript SDK](/build/sdks/ts-sdk)                            |
