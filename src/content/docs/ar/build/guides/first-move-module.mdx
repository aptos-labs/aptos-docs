في هذا البرنامج التعليمي، ستتعلم كيفية كتابة ونشر وحدة Move الأولى الخاصة بك على بلوك تشين أبتوس. Move هي لغة البرمجة الآمنة والمرنة المصممة خصيصاً للعقود الذكية على أبتوس.

سنقوم بإنشاء وحدة بسيطة للعدادات (Counter) تتيح للمستخدمين إنشاء عداد شخصي وزيادة قيمته.

<Aside type="note">
  هذا البرنامج التعليمي يفترض أن لديك [بيئة تطوير أبتوس مُعدة](/ar/build/get-started/developer-setup) مسبقاً.
</Aside>

## ما ستتعلمه

- كيفية إنشاء مشروع Move جديد
- فهم هيكل وحدة Move الأساسية
- كتابة وظائف للقراءة والكتابة
- تجميع واختبار الكود
- نشر الوحدة على بلوك تشين أبتوس
- التفاعل مع الوحدة المنشورة

## 1. إعداد المشروع

<Steps>

  3. **فحص ملف Move.toml:**
     ```toml
     [package]

     [dev-addresses]

     git = "https://github.com/aptos-labs/aptos-core.git"
     rev = "mainnet"
     subdir = "aptos-move/framework/aptos-framework"
     ```
</Steps>

## 2. كتابة وحدة العداد

أنشئ ملف `sources/counter.move` واكتب الكود التالي:

```move filename="sources/counter.move"
module counter_addr::counter {
    use std::signer;
    use std::error;

    /// خطأ: العداد موجود بالفعل
    const ECOUNTER_ALREADY_EXISTS: u64 = 1;
    /// خطأ: العداد غير موجود
    const ECOUNTER_NOT_EXISTS: u64 = 2;

    /// مورد يمثل عداد فردي
    struct Counter has key {
        value: u64,
    }

    /// إنشاء عداد جديد للحساب
    public entry fun create_counter(account: &signer) {
        let account_addr = signer::address_of(account);
        
        // التأكد من عدم وجود عداد مسبقاً
        assert!(!exists<Counter>(account_addr), error::already_exists(ECOUNTER_ALREADY_EXISTS));
        
        // إنشاء العداد وتخزينه
        let counter = Counter { value: 0 };
        move_to(account, counter);
    }

    /// زيادة قيمة العداد بـ 1
    public entry fun increment(account: &signer) acquires Counter {
        let account_addr = signer::address_of(account);
        
        // التأكد من وجود العداد
        assert!(exists<Counter>(account_addr), error::not_found(ECOUNTER_NOT_EXISTS));
        
        // الحصول على مرجع قابل للتعديل للعداد
        let counter_ref = borrow_global_mut<Counter>(account_addr);
        counter_ref.value = counter_ref.value + 1;
    }

    /// زيادة قيمة العداد بمقدار محدد
    public entry fun increment_by(account: &signer, amount: u64) acquires Counter {
        let account_addr = signer::address_of(account);
        
        // التأكد من وجود العداد
        assert!(exists<Counter>(account_addr), error::not_found(ECOUNTER_NOT_EXISTS));
        
        // الحصول على مرجع قابل للتعديل للعداد
        let counter_ref = borrow_global_mut<Counter>(account_addr);
        counter_ref.value = counter_ref.value + amount;
    }

    /// قراءة قيمة العداد الحالية
    #[view]
    public fun get_count(addr: address): u64 acquires Counter {
        assert!(exists<Counter>(addr), error::not_found(ECOUNTER_NOT_EXISTS));
        borrow_global<Counter>(addr).value
    }

    /// التحقق من وجود عداد للحساب
    #[view]
    public fun counter_exists(addr: address): bool {
        exists<Counter>(addr)
    }

    /// إعادة تعيين العداد إلى الصفر
    public entry fun reset_counter(account: &signer) acquires Counter {
        let account_addr = signer::address_of(account);
        
        // التأكد من وجود العداد
        assert!(exists<Counter>(account_addr), error::not_found(ECOUNTER_NOT_EXISTS));
        
        // إعادة تعيين القيمة
        let counter_ref = borrow_global_mut<Counter>(account_addr);
        counter_ref.value = 0;
    }
}
```

## 3. فهم الكود

دعنا نشرح المكونات الرئيسية لوحدة العداد:

### هيكل Counter

```move
struct Counter has key {
    value: u64,
}
```

- `Counter` هو مورد (resource) يخزن قيمة العداد
- `has key` يعني أنه يمكن تخزينه في التخزين العالمي
- `value: u64` هو العدد الصحيح الذي يمثل قيمة العداد

### الوظائف العامة

1. **`create_counter`** - ينشئ عداد جديد للحساب
2. **`increment`** - يزيد العداد بـ 1
3. **`increment_by`** - يزيد العداد بمقدار محدد
4. **`get_count`** - يقرأ قيمة العداد (وظيفة view)
5. **`counter_exists`** - يتحقق من وجود عداد
6. **`reset_counter`** - يعيد تعيين العداد إلى الصفر

### أنواع الوظائف

- **`public entry`** - يمكن استدعاؤها من خارج الوحدة كمعاملة
- **`#[view]`** - وظائف للقراءة فقط، لا تغير الحالة
- **`acquires`** - تشير إلى أن الوظيفة تصل إلى مورد معين

## 4. كتابة الاختبارات

أنشئ ملف `tests/counter_tests.move`:

```move filename="tests/counter_tests.move"
#[test_only]
module counter_addr::counter_tests {
    use std::signer;
    use counter_addr::counter;

    #[test(account = @0x123)]
    fun test_create_counter(account: &signer) {
        let addr = signer::address_of(account);
        
        // إنشاء العداد
        counter::create_counter(account);
        
        // التحقق من وجود العداد
        assert!(counter::counter_exists(addr), 1);
        
        // التحقق من القيمة الأولية
        assert!(counter::get_count(addr) == 0, 2);
    }

    #[test(account = @0x123)]
    fun test_increment(account: &signer) {
        let addr = signer::address_of(account);
        
        // إنشاء العداد
        counter::create_counter(account);
        
        // زيادة العداد
        counter::increment(account);
        assert!(counter::get_count(addr) == 1, 1);
        
        // زيادة مرة أخرى
        counter::increment(account);
        assert!(counter::get_count(addr) == 2, 2);
    }

    #[test(account = @0x123)]
    fun test_increment_by(account: &signer) {
        let addr = signer::address_of(account);
        
        // إنشاء العداد
        counter::create_counter(account);
        
        // زيادة بمقدار 5
        counter::increment_by(account, 5);
        assert!(counter::get_count(addr) == 5, 1);
        
        // زيادة بمقدار 10
        counter::increment_by(account, 10);
        assert!(counter::get_count(addr) == 15, 2);
    }

    #[test(account = @0x123)]
    fun test_reset_counter(account: &signer) {
        let addr = signer::address_of(account);
        
        // إنشاء العداد وزيادته
        counter::create_counter(account);
        counter::increment_by(account, 42);
        assert!(counter::get_count(addr) == 42, 1);
        
        // إعادة تعيين العداد
        counter::reset_counter(account);
        assert!(counter::get_count(addr) == 0, 2);
    }

    #[test(account = @0x123)]
    #[expected_failure(abort_code = 0x80001, location = counter_addr::counter)]
    fun test_create_counter_twice_fails(account: &signer) {
        // إنشاء العداد مرتين يجب أن يفشل
        counter::create_counter(account);
        counter::create_counter(account); // يجب أن يفشل هنا
    }

    #[test(account = @0x123)]
    #[expected_failure(abort_code = 0x60002, location = counter_addr::counter)]
    fun test_increment_nonexistent_counter_fails(account: &signer) {
        // محاولة زيادة عداد غير موجود يجب أن تفشل
        counter::increment(account);
    }
}
```

## 6. نشر الوحدة

<Steps>
  1. **إعداد حساب أبتوس:**
     ```bash
     aptos init
     ```
     اختر `testnet` للتطوير.

  2. **تمويل الحساب:**
     ```bash
     aptos account fund-with-faucet --account default
     ```

  3. **نشر الوحدة:**

     ```bash
     aptos move publish --named-addresses counter_addr=default
     ```

     ستحتاج إلى تأكيد النشر بكتابة `yes`.

  4. **التحقق من النشر:**
     بعد النشر الناجح، يمكنك التحقق من الوحدة على [Aptos Explorer](https://explorer.aptoslabs.com/).
</Steps>

## 7. التفاعل مع الوحدة

الآن يمكنك التفاعل مع وحدتك المنشورة:

<Steps>
  1. **إنشاء عداد:**
     ```bash
     aptos move run \
       --function-id default::counter::create_counter \
       --assume-yes
     ```

  2. **زيادة العداد:**
     ```bash
     aptos move run \
       --function-id default::counter::increment \
       --assume-yes
     ```

  3. **قراءة قيمة العداد:**
     ```bash
     aptos move view \
       --function-id default::counter::get_count \
       --args address:default
     ```

  4. **زيادة العداد بمقدار محدد:**
     ```bash
     aptos move run \
       --function-id default::counter::increment_by \
       --args u64:5 \
       --assume-yes
     ```
</Steps>

## 8. التكامل مع التطبيقات

يمكنك أيضاً التفاعل مع وحدتك من خلال مجموعات التطوير:

<Tabs>
  <TabItem label="TypeScript">
    ```typescript
    import { Aptos, AptosConfig, Network, Account } from "@aptos-labs/ts-sdk";

    const config = new AptosConfig({ network: Network.TESTNET });
    const aptos = new Aptos(config);

    async function interactWithCounter(account: Account, moduleAddress: string) {
      // إنشاء عداد
      const createTransaction = await aptos.transaction.build.simple({
        sender: account.accountAddress,
        data: {
          function: `${moduleAddress}::counter::create_counter`,
          functionArguments: [],
        },
      });

      await aptos.signAndSubmitTransaction({
        signer: account,
        transaction: createTransaction,
      });

      // قراءة قيمة العداد
      const count = await aptos.view({
        payload: {
          function: `${moduleAddress}::counter::get_count`,
          functionArguments: [account.accountAddress],
        },
      });

      console.log("قيمة العداد الحالية:", count[0]);
    }
    ```
  </TabItem>

  <TabItem label="Python">
    ```python
    from aptos_sdk.client import RestClient
    from aptos_sdk.account import Account

    async def interact_with_counter(client: RestClient, account: Account, module_address: str):
        # إنشاء عداد
        create_payload = {
            "type": "entry_function_payload",
            "function": f"{module_address}::counter::create_counter",
            "arguments": [],
            "type_arguments": []
        }
        
        create_txn = await client.create_bcs_signed_transaction(account, create_payload)
        await client.submit_bcs_transaction(create_txn)

        # قراءة قيمة العداد
        count = await client.view_function(
            f"{module_address}::counter::get_count",
            [],
            [str(account.address())]
        )
        
        print(f"قيمة العداد الحالية: {count[0]}")
    ```
  </TabItem>
</Tabs>

## المفاهيم المتقدمة

```move
public entry fun increment_by_safe(account: &signer, amount: u64) acquires Counter {
    assert!(amount > 0, error::invalid_argument(EINVALID_AMOUNT));
    // باقي الكود...
}
```

### الأحداث (Events)

```move
use std::event;

struct CounterIncremented has drop, store {
    old_value: u64,
    new_value: u64,
}

public entry fun increment_with_event(account: &signer) acquires Counter {
    let account_addr = signer::address_of(account);
    let counter_ref = borrow_global_mut<Counter>(account_addr);
    
    let old_value = counter_ref.value;
    counter_ref.value = counter_ref.value + 1;
    
    // إصدار حدث
    event::emit(CounterIncremented {
        old_value,
        new_value: counter_ref.value,
    });
}
```
