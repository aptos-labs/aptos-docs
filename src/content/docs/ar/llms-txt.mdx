## ما هو LLMs.txt؟

نحن ندعم ملفات [LLMs.txt](https://llmstxt.org/) لجعل وثائق Aptos متاحة للنماذج اللغوية الكبيرة
(LLMs). هذه الميزة تساعد أدوات الذكاء الاصطناعي على فهم بلوك تشين Aptos ولغة Move وSDKs وأنماط التطوير بشكل أفضل.

## المسارات المتاحة

نحن نوفر المسارات التالية لـ LLMs.txt لمساعدة أدوات الذكاء الاصطناعي في الوصول إلى وثائقنا:

- [llms.txt](/llms.txt) - يحتوي على نظرة عامة منظمة لملفات Aptos LLMs.txt
- [llms-small.txt](/llms-small.txt) - يوفر نسخة مكثفة من الوثائق، محسنة للنوافذ السياقية الأصغر
- [llms-full.txt](/llms-full.txt) - يوفر الوثائق الشاملة الكاملة لجميع مفاهيم Aptos

## الاستخدام مع أدوات الذكاء الاصطناعي

### Cursor

استخدم ميزة `@Docs` في Cursor لتضمين ملفات LLMs.txt في مشروعك. هذا يساعد Cursor على توفير اقتراحات
كود ووثائق أكثر دقة لتطوير Aptos.

[اقرأ المزيد حول @Docs في Cursor](https://docs.cursor.com/context/@-symbols/@-docs)

### GitHub Copilot

يمكن لـ GitHub Copilot الاستفادة من المعلومات في ملفات LLMs.txt هذه لتوفير مساعدة أفضل عند تطوير تطبيقات Aptos. يمكنك الإشارة إلى هذه الملفات في GitHub Copilot Chat باستخدام الروابط التالية:

```
https://aptos.dev/llms.txt
https://aptos.dev/llms-small.txt
https://aptos.dev/llms-full.txt
```

### Windsurf

أشر إلى ملفات LLMs.txt باستخدام `@` أو في ملفات `.windsurfrules` الخاصة بك لتحسين فهم Windsurf لتطوير Aptos.

[اقرأ المزيد حول Windsurf Memories](https://docs.codeium.com/windsurf/memories#memories-and-rules)

### أدوات الذكاء الاصطناعي الأخرى

أي أداة ذكاء اصطناعي تدعم LLMs.txt يمكنها استخدام هذه المسارات لفهم Aptos بشكل أفضل. ببساطة وجه أداتك إلى أي من
المسارات المذكورة أعلاه.
