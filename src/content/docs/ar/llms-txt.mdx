## ما هو LLMs.txt؟

نحن ندعم ملفات [LLMs.txt](https://llmstxt.org/) لجعل وثائق Aptos متاحة للنماذج اللغوية الكبيرة (LLMs). تساعد هذه الميزة أدوات الذكاء الاصطناعي على فهم بلوكشين Aptos ولغتها Move وSDKs وأنماط التطوير بشكل أفضل.

## المسارات المتاحة

نحن نقدم مسارات LLMs.txt التالية لمساعدة أدوات الذكاء الاصطناعي على الوصول إلى وثائقنا:

- [llms.txt](/llms.txt) - يحتوي على نظرة عامة منظمة لملفات LLMs.txt الخاصة بـ Aptos
- [llms-small.txt](/llms-small.txt) - يوفر نسخة مختصرة من الوثائق، محسنة للنوافذ السياقية الأصغر
- [llms-full.txt](/llms-full.txt) - يوفر الوثائق الكاملة والشاملة لجميع مفاهيم Aptos

## الاستخدام مع أدوات الذكاء الاصطناعي

### Cursor

استخدم ميزة `@Docs` في Cursor لتضمين ملفات LLMs.txt في مشروعك. يساعد ذلك Cursor على تقديم اقتراحات كود أكثر دقة ووثائق لتطوير Aptos.

[اقرأ المزيد عن @Docs في Cursor](https://docs.cursor.com/context/@-symbols/@-docs)

### GitHub Copilot

يمكن لـ GitHub Copilot الاستفادة من المعلومات الموجودة في ملفات LLMs.txt هذه لتقديم مساعدة أفضل عند تطوير تطبيقات Aptos. يمكنك الرجوع إلى هذه الملفات في GitHub Copilot Chat باستخدام عناوين URL التالية:

```
https://aptos.dev/llms.txt
https://aptos.dev/llms-small.txt
https://aptos.dev/llms-full.txt
```

### Windsurf

راجع ملفات LLMs.txt باستخدام `@` أو في ملفات `.windsurfrules` لتحسين فهم Windsurf لتطوير Aptos.

[اقرأ المزيد عن Windsurf Memories](https://docs.codeium.com/windsurf/memories#memories-and-rules)

### أدوات الذكاء الاصطناعي الأخرى

أي أداة ذكاء اصطناعي تدعم LLMs.txt يمكنها استخدام هذه المسارات لفهم Aptos بشكل أفضل. فقط وجه أداتك إلى أي من المسارات أعلاه.
