Das Aptos-Blockchain-Netzwerk kann über mehrere APIs zugänglich gemacht werden, je nach Ihrem Anwendungsfall.

## Aptos Fullnode

Diese API - in Fullnodes eingebettet - bietet eine einfache, latenzarme und dennoch low-level Möglichkeit, den Status zu _lesen_ und Transaktionen an die Aptos-Blockchain zu _senden_. Sie unterstützt auch Transaktionssimulation.

Der Code jeder der oben genannten APIs ist Open Source auf [GitHub](https://github.com/aptos-labs/aptos-core). Als solche kann jeder diese APIs betreiben und viele unabhängige Betreiber und Entwickler weltweit entscheiden sich dafür.

### Von Aptos Labs betriebene API-Bereitstellungen

[Aptos Labs](https://aptoslabs.com) betreibt im Namen der [Aptos Foundation](https://aptosfoundation.org/) eine Bereitstellung dieser APIs für jedes [Aptos-Netzwerk](/de/network/nodes/networks) und stellt sie für den öffentlichen Verbrauch zur Verfügung.

Diese APIs ermöglichen begrenzten Zugang auf IP-Basis ohne einen API-Schlüssel (anonymer Zugang). Um viel höhere Ratenlimits zu erhalten, können Sie sich für ein [Aptos Build](https://build.aptoslabs.com/)-Konto anmelden.
