Die Aptos-Befehlszeilenschnittstelle (CLI) ist ein Tool, das Ihnen hilft, Move-Verträge zu kompilieren und zu testen. Sie kann Ihnen auch dabei helfen, schnell mit Aptos-Funktionen auf der Kette zu experimentieren.

Für fortgeschrittene Benutzer kann die CLI auch verwendet werden, um ein privates Aptos-Netzwerk zu betreiben (um Code lokal zu testen) und kann bei der Verwaltung eines Netzwerkknotens hilfreich sein.
