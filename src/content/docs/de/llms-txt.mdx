## Was ist LLMs.txt?

Wir unterstützen [LLMs.txt](https://llmstxt.org/)-Dateien, um die Aptos-Dokumentation für große Sprachmodelle
(LLMs) verfügbar zu machen. Diese Funktion hilft KI-Tools dabei, die Aptos-Blockchain, ihre Move-Sprache, SDKs und Entwicklungsmuster besser zu verstehen.

## Verfügbare Routen

Wir stellen die folgenden LLMs.txt-Routen zur Verfügung, um KI-Tools den Zugang zu unserer Dokumentation zu erleichtern:

- [llms.txt](/llms.txt) - Enthält eine strukturierte Übersicht über Aptos LLMs.txt-Dateien
- [llms-small.txt](/llms-small.txt) - Bietet eine komprimierte Version der Dokumentation, optimiert für kleinere Kontextfenster
- [llms-full.txt](/llms-full.txt) - Bietet die vollständige umfassende Dokumentation für alle Aptos-Konzepte

## Verwendung mit KI-Tools

### Cursor

Verwenden Sie die `@Docs`-Funktion in Cursor, um die LLMs.txt-Dateien in Ihr Projekt einzubinden. Das hilft Cursor dabei, genauere
Code-Vorschläge und Dokumentation für die Aptos-Entwicklung bereitzustellen.

[Mehr über @Docs in Cursor lesen](https://docs.cursor.com/context/@-symbols/@-docs)

### GitHub Copilot

GitHub Copilot kann die Informationen in diesen LLMs.txt-Dateien nutzen, um bessere Unterstützung bei der Entwicklung von Aptos-Anwendungen zu bieten. Sie können auf diese Dateien in Ihrem GitHub Copilot Chat verweisen, indem Sie die folgenden URLs verwenden:

```
https://aptos.dev/llms.txt
https://aptos.dev/llms-small.txt
https://aptos.dev/llms-full.txt
```

### Windsurf

Verweisen Sie auf die LLMs.txt-Dateien mit `@` oder in Ihren `.windsurfrules`-Dateien, um Windsurfs Verständnis für die Aptos-Entwicklung zu verbessern.

[Mehr über Windsurf Memories lesen](https://docs.codeium.com/windsurf/memories#memories-and-rules)

### Andere KI-Tools

Jedes KI-Tool, das LLMs.txt unterstützt, kann diese Routen verwenden, um Aptos besser zu verstehen. Richten Sie Ihr Tool einfach auf eine der
oben genannten Routen.
