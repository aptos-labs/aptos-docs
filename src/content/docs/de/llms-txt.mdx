## Was ist LLMs.txt?

Wir unterstützen [LLMs.txt](https://llmstxt.org/)-Dateien, um die Aptos-Dokumentation für große Sprachmodelle (LLMs) verfügbar zu machen. Diese Funktion hilft KI-Tools, die Aptos-Blockchain, ihre Move-Sprache, SDKs und Entwicklungs-Pattern besser zu verstehen.

## Verfügbare Routen

Wir bieten die folgenden LLMs.txt-Routen, um KI-Tools beim Zugriff auf unsere Dokumentation zu helfen:

- [llms.txt](/llms.txt) - Enthält eine strukturierte Übersicht der Aptos LLMs.txt-Dateien
- [llms-small.txt](/llms-small.txt) - Bietet eine kondensierte Version der Dokumentation, optimiert für kleinere Kontextfenster
- [llms-full.txt](/llms-full.txt) - Bietet die vollständige umfassende Dokumentation für alle Aptos-Konzepte

## Verwendung mit KI-Tools

### Cursor

Verwenden Sie die `@Docs`-Funktion in Cursor, um die LLMs.txt-Dateien in Ihr Projekt einzuschließen. Dies hilft Cursor, genauere Codevorschläge und Dokumentation für die Aptos-Entwicklung zu liefern.

[Mehr über @Docs in Cursor lesen](https://docs.cursor.com/context/@-symbols/@-docs)

### GitHub Copilot

GitHub Copilot kann die Informationen in diesen LLMs.txt-Dateien nutzen, um bessere Unterstützung bei der Entwicklung von Aptos-Anwendungen zu bieten. Sie können auf diese Dateien in Ihrem GitHub Copilot Chat mit den folgenden URLs verweisen:

```
https://aptos.dev/llms.txt
https://aptos.dev/llms-small.txt
https://aptos.dev/llms-full.txt
```

### Windsurf

Verweisen Sie auf die LLMs.txt-Dateien mit `@` oder in Ihren `.windsurfrules`-Dateien, um das Verständnis von Windsurf für die Aptos-Entwicklung zu verbessern.

[Mehr über Windsurf Memories lesen](https://docs.codeium.com/windsurf/memories#memories-and-rules)

### Andere KI-Tools

Jedes KI-Tool, das LLMs.txt unterstützt, kann diese Routen verwenden, um Aptos besser zu verstehen. Richten Sie einfach Ihr Tool auf eine der Routen oben.
