Der Aptos Testnet Faucet ermöglicht es Entwicklern, kostenlose APT-Token im **Aptos testnet** instant anzufordern. Diese Token sind nur für **Testzwecke** gedacht - sie haben keinen realen Wert und können nicht gegen mainnet APT eingetauscht werden.

Verwenden Sie den Faucet für:

- **Move Smart Contracts** bereitstellen und testen
- Mit Aptos SDKs experimentieren (TypeScript, Python, Rust und mehr)
- dApps vor dem Launch ins mainnet aufbauen und verfeinern

<Faucet />

## Wie man den Faucet verwendet

1. **Melden Sie sich mit Ihrem Google-Konto an**.
2. **Geben Sie Ihre Aptos testnet Wallet-Adresse ein**.
3. Klicken Sie auf **APT anfordern**, um instant Token zu erhalten.
4. Überprüfen Sie Ihr Wallet-Guthaben - Token sollten innerhalb weniger Sekunden erscheinen.

> **Tipp:** Wenn Sie kein testnet-Konto haben, können Sie entweder [Ein Konto mit dem Aptos TypeScript SDK erstellen](/build/sdks/ts-sdk/quickstart) (Entwickler-Methode) oder [Eine Aptos Wallet installieren und zum Testnet wechseln](https://aptosfoundation.org/ecosystem/projects/wallets) (UI-Methode), um zu starten.
