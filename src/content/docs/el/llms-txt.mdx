## Τι είναι το LLMs.txt;

Υποστηρίζουμε αρχεία [LLMs.txt](https://llmstxt.org/) για να κάνουμε την τεκμηρίωση του Aptos διαθέσιμη σε μεγάλα γλωσσικά μοντέλα
(LLMs). Αυτή η λειτουργία βοηθά τα εργαλεία AI να κατανοήσουν καλύτερα το blockchain Aptos, τη γλώσσα Move, τα SDKs και τα μοτίβα ανάπτυξης.

## Διαθέσιμες Διαδρομές

Παρέχουμε τις ακόλουθες διαδρομές LLMs.txt για να βοηθήσουμε τα εργαλεία AI να αποκτήσουν πρόσβαση στην τεκμηρίωσή μας:

- [llms.txt](/llms.txt) - Περιέχει μια δομημένη επισκόπηση των αρχείων LLMs.txt του Aptos
- [llms-small.txt](/llms-small.txt) - Παρέχει μια συμπυκνωμένη εκδοχή της τεκμηρίωσης, βελτιστοποιημένη για μικρότερα παράθυρα περιβάλλοντος
- [llms-full.txt](/llms-full.txt) - Παρέχει την πλήρη περιεκτική τεκμηρίωση για όλες τις έννοιες του Aptos

## Χρήση με Εργαλεία AI

### Cursor

Χρησιμοποιήστε τη λειτουργία `@Docs` στο Cursor για να συμπεριλάβετε τα αρχεία LLMs.txt στο έργο σας. Αυτό βοηθά το Cursor να παρέχει πιο ακριβείς
προτάσεις κώδικα και τεκμηρίωση για την ανάπτυξη Aptos.

[Διαβάστε περισσότερα για το @Docs στο Cursor](https://docs.cursor.com/context/@-symbols/@-docs)

### GitHub Copilot

Το GitHub Copilot μπορεί να αξιοποιήσει τις πληροφορίες σε αυτά τα αρχεία LLMs.txt για να παρέχει καλύτερη βοήθεια κατά την ανάπτυξη εφαρμογών Aptos. Μπορείτε να αναφέρετε αυτά τα αρχεία στο GitHub Copilot Chat χρησιμοποιώντας τα ακόλουθα URLs:

```
https://aptos.dev/llms.txt
https://aptos.dev/llms-small.txt
https://aptos.dev/llms-full.txt
```

### Windsurf

Αναφέρετε τα αρχεία LLMs.txt χρησιμοποιώντας `@` ή στα αρχεία `.windsurfrules` για να ενισχύσετε την κατανόηση του Windsurf για την ανάπτυξη Aptos.

[Διαβάστε περισσότερα για τις Μνήμες Windsurf](https://docs.codeium.com/windsurf/memories#memories-and-rules)

### Άλλα Εργαλεία AI

Οποιοδήποτε εργαλείο AI που υποστηρίζει LLMs.txt μπορεί να χρησιμοποιήσει αυτές τις διαδρομές για να κατανοήσει καλύτερα το Aptos. Απλά κατευθύνετε το εργαλείο σας σε οποιαδήποτε από τις
παραπάνω διαδρομές.
