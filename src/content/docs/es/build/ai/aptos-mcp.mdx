# Protocolo de Contexto de Modelo de Aptos (MCP)

El Protocolo de Contexto de Modelo de Aptos (MCP) es un servidor que proporciona un conjunto de herramientas, prompts y recursos para ayudar a los desarrolladores a construir aplicaciones en la blockchain de Aptos. Está diseñado para ser usado con herramientas de IA como Cursor, Claude Code y otras que soportan el Protocolo de Contexto de Modelo.

## Comenzando

### Prerequisitos

- [node y npm](https://nodejs.org/en)
- Clave API de Build Bot

### Generar una Clave API de Build Bot

Para poder realizar acciones de Geomi como gestionar claves API, etc., sigue estas instrucciones para generar una nueva Clave API de Bot para usar con el MCP.

- Ve a https://geomi.dev/
- Haz clic en tu nombre en la esquina inferior izquierda
- Haz clic en "Bot Keys"
- Haz clic en el botón "Create Bot Key"
- Copia la Clave de Bot y pégala en el archivo de configuración del MCP como un argumento env: `APTOS_BOT_KEY=<tu-clave-de-bot>`

### Interfaces Soportadas

Hemos proporcionado guías para Cursor y Claude Code para ayudarte a integrar el Aptos MCP en tu entorno de desarrollo. Si estás usando una herramienta de IA diferente, sigue los pasos para tu herramienta de IA favorita y consulta la documentación para Cursor o Claude Code como ejemplos.
