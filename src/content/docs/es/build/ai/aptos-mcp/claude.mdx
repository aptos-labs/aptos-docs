## Configurar Claude Code con Aptos MCP

1. Instala el paquete `claude-code`

```bash
npm install -g @anthropic-ai/claude-code
```

2. Localiza donde Claude Code almacena su configuración, usualmente en Mac está en `~/.claude.json`
3. Edita el objeto `mcpServers` en el archivo `json` con

```json
{
  "mcpServers": {
    "aptos-mcp": {
      "command": "npx",
      "args": ["-y", "@aptos-labs/aptos-mcp"],
      "type": "stdio",
      "env": {
        "APTOS_BOT_KEY": "<clave_api_bot>"
      }
    }
  }
}
```

4. Obtén tu `APTOS_BOT_KEY`:

- Visita [Geomi](https://geomi.dev/) e inicia sesión con tu cuenta.
- Navega a la sección de Claves API y crea una nueva clave.
- Copia la clave generada para usar en el siguiente paso.

5. Asegúrate de actualizar la `APTOS_BOT_KEY` con la clave que generaste en el paso anterior.

6. Navega a tu proyecto

```bash
cd tu-proyecto-increible
```

7. En una nueva ventana de terminal escribe:

```bash
claude
```

8. Ahora puedes usar Claude Code para interactuar con el Aptos MCP. Pregunta al agente con `¿qué versión de aptos mcp estás usando?` para verificar la conexión. El agente debería responder con algo como:

```text
Estoy usando Aptos MCP versión 0.0.2.
```
