# Configurar Cursor con Aptos MCP

1. Abre el IDE de Cursor
2. En la carpeta raíz del proyecto, crea una carpeta `.cursor`
3. En la carpeta `.cursor`, crea un archivo `mcp.json`
4. Pega este contenido

```json
{
  "mcpServers": {
    "aptos-mcp": {
      "command": "npx",
      "args": ["-y", "@aptos-labs/aptos-mcp"],
      "env": {
        "APTOS_BOT_KEY": "<clave_api_bot>"
      }
    }
  }
}
```

5. Obtén tu `APTOS_BOT_KEY`:
   - Visita [Geomi](https://geomi.dev/) e inicia sesión con tu cuenta.
   - Navega a la sección de Claves API y genera una nueva clave.
   - Copia la clave generada para usar en el siguiente paso.

6. Asegúrate de actualizar la `APTOS_BOT_KEY` en el archivo `mcp.json` con la clave que acabas de generar.

### Verificar que Cursor ejecute tu MCP

1. Abre la Configuración de Cursor: `cursor -> settings -> cursor settings`
2. Ve a la sección `MCP` o `Tools & Integrations`
3. Asegúrate de que esté habilitado y mostrando un indicador de color verde
