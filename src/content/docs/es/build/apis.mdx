La red de Blockchain de Aptos puede ser accedida por varias APIs, dependiendo de tu caso de uso.

## Nodo Completo de Aptos

Esta API - integrada en Nodos Completos - proporciona una forma simple, de baja latencia, pero de bajo nivel de _leer_ estado y
_enviar_ transacciones a la Blockchain de Aptos. También soporta simulación de transacciones.

El código de cada una de las APIs mencionadas anteriormente es de código abierto en [GitHub](https://github.com/aptos-labs/aptos-core). Como
tal, cualquiera puede operar estas APIs y muchos operadores y constructores independientes en todo el mundo eligen hacerlo.

### Despliegues de API Operados por Aptos Labs

[Aptos Labs](https://aptoslabs.com) opera un despliegue de estas APIs en nombre de [Aptos Foundation](https://aptosfoundation.org/)
para cada [Red de Aptos](/es/network/nodes/networks) y las hace disponibles para consumo público.

Estas APIs permiten acceso limitado por IP sin una clave de API (acceso anónimo). Para obtener límites de tasa mucho más altos puedes registrarte para una cuenta de [Geomi](https://geomi.dev/).
