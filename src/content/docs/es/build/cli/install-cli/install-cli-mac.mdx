Para Mac, la forma más fácil de instalar la CLI de Aptos es con el gestor de paquetes `brew`.

# Instalación

<Steps>
  1. Asegúrate de tener brew instalado https://brew.sh/.

  2. Abre una nueva terminal e ingresa los siguientes comandos.

     ```shellscript filename="Terminal"
     brew update
     brew install aptos
     ```

  3. Abre otra terminal y ejecuta aptos help para verificar que la CLI esté instalada.

     ```shellscript filename="Terminal"
     aptos help
     ```
</Steps>

<Aside type="caution">
  Si `brew` no funciona para ti, puedes probar los pasos aquí: [Instalar mediante Script](#install-via-script) o [Instalar mediante Binarios Precompilados](#install-via-pre-compiled-binaries-backup-method).)
</Aside>

# Actualizar la CLI

Actualizar la CLI con brew solo toma 2 comandos:

```shellscript filename="Terminal"
brew update
brew upgrade aptos
```

# Instalar mediante Script

<Steps>
  1. En la terminal, usa uno de los siguientes comandos:

     ```shellscript filename="Terminal"
     curl -fsSL "https://aptos.dev/scripts/install_cli.sh" | sh
     ```

     O usa el comando equivalente `wget`:

     ```shellscript filename="Terminal"
     wget -qO- "https://aptos.dev/scripts/install_cli.sh" | sh
     ```

  2. (Opcional) Puede ser útil agregar la CLI de Aptos a una carpeta en tu PATH, o agregarla directamente a tu PATH.

     - Los pasos para agregar una carpeta a tu PATH dependen del shell.
     - Puedes ejecutar `echo $SHELL` para imprimir el shell predeterminado de tu máquina, luego buscar en Google los pasos específicos para agregar una carpeta a tu PATH para ese shell.

  3. Verifica que el script esté instalado abriendo una nueva terminal y ejecutando aptos help

     - Deberías ver una lista de comandos que puedes ejecutar usando la CLI.
     - En el futuro, este es un recurso útil para aprender exactamente cómo funciona cada comando.
</Steps>

<Aside type="note" emoji="ℹ️">
  Si deseas actualizar la CLI de Aptos a la última versión, puedes ejecutar `aptos update`.
</Aside>

# Instalar mediante Binarios Precompilados (Método de Respaldo)

<Steps>
  1. Ve a la [página de releases de GitHub](https://github.com/aptos-labs/aptos-core/releases).

  2. Haz clic en el menú expandible "Assets" para la última versión para ver los binarios precompilados.

  3. Descarga el archivo zip para macOS.

     1. Tendrá un nombre como: `aptos-cli-<version>-macOS-x86_64.zip` o `aptos-cli-<version>-macOS-arm64.zip`.
     2. Asegúrate de elegir el archivo zip correcto para la arquitectura de tu computadora (x86\_64 para Intel / AMD o arm64 para ARM).
     3. Es probable que tengas que descartar advertencias de que este es un archivo sospechoso al descargarlo.

  4. Descomprime el archivo descargado.

  5. Mueve el archivo binario de Aptos extraído a tu carpeta preferida.

  6. Abre una terminal y navega a tu carpeta preferida.

  7. Haz que ~/aptos sea ejecutable ejecutando chmod +x ~/aptos.

  8. Verifica que esta versión instalada funcione ejecutando ~/aptos help.

     Deberías ver instrucciones sobre cómo usar todos los comandos de la CLI. Estos pueden ser útiles en el futuro cuando estés tratando de entender cómo usar comandos específicos.

  9. (Opcional) Puede ser útil agregar la CLI de Aptos a una carpeta en tu PATH, o agregarla directamente a tu PATH.

     - Los pasos para agregar una carpeta a tu PATH dependen del shell.
     - Puedes ejecutar `echo $SHELL` para imprimir el shell predeterminado de tu máquina, luego buscar en Google los pasos específicos para agregar una carpeta a tu PATH para ese shell.
</Steps>

<Aside type="note" emoji="ℹ️">
  Al usar el método de binarios precompilados, puedes actualizar la CLI de Aptos eliminando tu instalación existente, luego siguiendo los pasos de instalación nuevamente.
</Aside>
