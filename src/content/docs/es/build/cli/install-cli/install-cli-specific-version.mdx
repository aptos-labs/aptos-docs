---
title: "Instalar Versiones Específicas de la CLI de Aptos (Avanzado)"
---

import { Steps } from '@astrojs/starlight/components';

Si necesitas una versión específica de la CLI de Aptos, puedes compilarla directamente desde el código fuente de Aptos. Este método de instalación se usa principalmente para interactuar con características específicas en Devnet que pueden no haber llegado a Testnet / Mainnet aún. También puedes querer seguir estos pasos si estás ejecutando una arquitectura que no funciona bien con las versiones existentes / binarios precompilados.

Si no necesitas este método avanzado, puedes encontrar los pasos de instalación normales [aquí](/build/cli).

## Instalar en macOS / Linux

<Steps>
  1. Sigue los pasos para .

  2. Asegúrate de tener cargo instalado siguiendo los pasos en .

  3. Compila la herramienta CLI: cargo build --package aptos --profile cli.

     El binario estará disponible en `target/cli/aptos`.

  4. (Opcional) Mueve este ejecutable a un lugar en tu PATH.

  5. Verifica que la instalación funcionó ejecutando target/cli/aptos help.

     Estas instrucciones de ayuda también sirven como una guía detallada útil para comandos específicos.
</Steps>

## Instalar en Windows

<Steps>
  1. Sigue los pasos para compilar Aptos desde el código fuente .

  2. Asegúrate de tener cargo instalado siguiendo los pasos en .

  3. Compila la herramienta CLI: cargo build --package aptos --profile cli.

     El binario estará disponible en `target\cli\aptos.exe`.

  4. (Opcional) Mueve este ejecutable a un lugar en tu PATH.

  5. Verifica que la instalación funcionó ejecutando target\cli\aptos.exe help.

     Estas instrucciones de ayuda también sirven como una guía detallada útil para comandos específicos.
</Steps>
