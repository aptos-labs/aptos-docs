---
title: "Gestionar un Nodo de Red vía la CLI de Aptos"
---

Si estás ejecutando un [nodo validador o nodo completo validador (VFN)](/network/nodes/validator-node), puedes usar la CLI para interactuar con tu nodo.

Específicamente, puedes usar la CLI para:

1. [Gestionar pools de staking que posees](/network/nodes/validator-node/connect-nodes/staking-pool-operations).
2. [Votar en propuestas](/network/nodes/validator-node/connect-nodes/staking-pool-voter).

Más allá de eso, puedes ejecutar este comando de ayuda para ver más comandos especializados que la CLI puede hacer relacionados con operar tu nodo:

```shellscript filename="Terminal"
aptos node --help
```
