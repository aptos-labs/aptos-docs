Si estás ejecutando un [nodo validador o nodo completo validador (VFN)](/es/network/nodes/validator-node), puedes usar el CLI para interactuar con tu nodo.

Específicamente, puedes usar el CLI para:

1. [Gestionar pools de staking que posees](/es/network/nodes/validator-node/connect-nodes/staking-pool-operations).
2. [Votar en propuestas](/es/network/nodes/validator-node/connect-nodes/staking-pool-voter).

Más allá de eso, puedes ejecutar este comando de ayuda para ver comandos más especializados que el CLI puede hacer relacionados con operar tu nodo:

```shellscript filename="Terminal"
aptos node --help
```
