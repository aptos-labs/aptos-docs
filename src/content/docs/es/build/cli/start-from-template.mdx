Sigue los pasos a continuación para comenzar rápidamente.

<Steps>
  1. Inicializar

     Ejecuta lo siguiente para inicializar un paquete usando la plantilla `hello-blockchain`:

     ```shellscript filename="Terminal"
     aptos move init --name hello_blockchain --template hello-blockchain
     ```

  2. Comenzar a construir

     La plantilla crea un archivo `hello_blockchain.move` bajo `sources` para ayudarte a comenzar.

     <RemoteCodeblock permalink="https://github.com/aptos-labs/aptos-core/blob/afd3706c17bcccfb39a9d6059aecbfa648ed295d/aptos-move/move-examples/hello_blockchain/sources/hello_blockchain.move#L1-L64" />

  3. Ver todas las plantillas

     Ejecuta el siguiente comando para ver todas las plantillas (y para ayuda general al inicializar un paquete):

     ```shellscript
     aptos move init --help
     ```
</Steps>
