El CLI puede ser una herramienta conveniente para buscar rápidamente datos en cadena y enviar transacciones desde tus cuentas.

La forma más común de especificar con qué cuentas quieres interactuar es a través de perfiles. Puedes crear un nuevo perfil en el cli ejecutando el siguiente comando:

```shellscript filename="Terminal"
aptos init --profile <nombre-de-tu-perfil>
```

Si algún comando toma una cuenta, puedes pasar el nombre de un perfil en su lugar. Si un comando usa implícitamente el perfil predeterminado, usualmente tendrá un parámetro opcional para usar un perfil específico en su lugar que puedes encontrar ejecutando `aptos <tu-comando> --help`.
