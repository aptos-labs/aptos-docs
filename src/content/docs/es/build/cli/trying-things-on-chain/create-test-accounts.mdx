---
title: "Crear Cuentas de Prueba y Enviar Transacciones desde la CLI de Aptos"
---

import { Aside } from '@astrojs/starlight/components';

<Aside type="note" emoji="ℹ️">
  Puedes instalar la CLI de Aptos siguiendo [estos pasos](/build/cli) si aún no lo has hecho.
</Aside>

En general, para crear una nueva cuenta en la cadena, necesitarás generar claves y luego financiar la cuenta. En devnet, puedes financiar una nueva cuenta pidiéndole a una cuenta "faucet" con tokens de prueba de Aptos que los envíe a tu cuenta. En testnet puedes acuñar en la [página de acuñación](/network/faucet).

Usando la CLI, puedes generar y financiar una cuenta de prueba usando:

```shellscript filename="Terminal"
aptos init --profile <tu-nombre-de-perfil>
```

Una vez que tengas una cuenta financiada, puedes enviar monedas entre cuentas con el comando `transfer` así:

```shellscript filename="Terminal"
aptos account transfer --account superuser --amount 100
```

<Aside type="note" emoji="ℹ️">
  Para tener más control sobre cómo se ven tus credenciales generadas, en lugar de `aptos init`, puedes usar:

  1. `aptos key generate --vanity-prefix 0x<tu-prefijo>`
  2. `aptos account fund-with-faucet --account <tu-dirección-de-cuenta-recién-generada>`

  Ten en cuenta sin embargo que las direcciones son diferentes a las claves.
</Aside>
