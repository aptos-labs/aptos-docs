`getNumberOfDelegators` recupera el conteo de delegadores activos para una dirección de pool especificada. Esta consulta es especialmente valiosa para plataformas que necesitan analizar la participación en pools de staking o delegación.

<Aside type="note">
  ¡Pruébalo ahora! Personaliza la variable de abajo en el editor para explorar diferentes direcciones de pool.
</Aside>

<GraphQLEditor
  query={`query getNumberOfDelegators($poolAddress: String) {
  num_active_delegator_per_pool(
    where: {
      pool_address: { _eq: $poolAddress }
      num_active_delegator: { _gt: "0" }
    }
    distinct_on: pool_address
  ) {
    num_active_delegator
  }
}`}
  variables={`{
  "poolAddress": "0x06099edbe54f242bad50020dfd67646b1e46282999483e7064e70f02f7ea3c15"
}`}
/>

#### Variables:

- `$poolAddress`: **String** - La dirección del pool para el cual consultar el número de delegadores activos. Ejemplo: `"0x06099edbe54f242bad50020dfd67646b1e46282999483e7064e70f02f7ea3c15"`.
