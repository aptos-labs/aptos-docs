**AptosKit** es un SDK Swift para interactuar con la blockchain de Aptos.
Proporciona una interfaz simple y fácil de usar para interactuar con la
blockchain de Aptos.

El SDK es una exportación de paquete Swift para [Kaptos](/es/build/sdks/community-sdks/kotlin-sdk), un SDK Kotlin para
la blockchain de Aptos. Se construye haciendo su binario iOS disponible como una
dependencia para desarrolladores iOS trabajando en proyectos Swift nativos.

<Aside type="note">
  **AptosKit** está actualmente en desarrollo, por favor da retroalimentación [aquí](https://github.com/mcxross/swift-aptos/issues)
</Aside>

## Instalación

Para instalar **AptosKit**, agrega lo siguiente a tu archivo `Package.swift`:

```swift filename="Package.swift"
dependencies: [
    .package(url: "https://github.com/mcxross/swift-aptos.git", .upToNextMajor(from: <version>))
]
```
