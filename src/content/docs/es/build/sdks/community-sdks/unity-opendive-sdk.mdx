El [SDK Unity de Aptos](https://github.com/aptos-labs/Aptos-Unity-SDK) es una implementación .NET del [SDK de Aptos](/es/build/sdks), compatible con .NET Standard 2.0 y .NET 4.x para Unity. El objetivo de este SDK es proporcionar un conjunto de herramientas para que los desarrolladores construyan aplicaciones multiplataforma (móvil, escritorio, web, VR) usando el motor de juego Unity y la infraestructura blockchain de Aptos.

Ver la publicación [Aptos Labs brings Web3 to Gaming with its new SDK for Unity developers](https://medium.com/aptoslabs/aptos-labs-brings-web3-to-gaming-with-its-new-sdk-for-unity-developers-e6544bdf9ba9) y la sección [Detalles técnicos](https://github.com/aptos-labs/Aptos-Unity-SDK#technical-details) del README del SDK Unity para todas las características ofrecidas a los desarrolladores de juegos por el SDK Unity de Aptos.

## Flujos de usuario

El SDK Unity de Aptos soporta estos casos de uso:

- _Flujo de incorporación progresiva_ en el que los usuarios pueden iniciar sesión en un juego por email. En este flujo, las transacciones son proxied, y Aptos usa un sistema de claves distribuidas. Los usuarios pueden luego incorporarse a una billetera custodial completa si lo desean.
- _Integración de billetera no-custodial en el juego_ en la que los desarrolladores de juegos tienen la opción de permitir a los usuarios crear billeteras no-custodiales completas en los juegos.
- _Integración de billetera no-custodial fuera del juego_ en la que los desarrolladores de juegos pueden permitir a los usuarios conectarse a una billetera de escritorio o una billetera móvil dentro del juego o crear billeteras burner desde la billetera padre sin problemas.

## Instalar el SDK Unity

Puedes instalar el SDK Unity ya sea a través de nuestro `unitypackage` o el [Unity Package Manager](https://docs.unity3d.com/Manual/Packages.html).

### Instalar por `unitypackage`

1. Inicia Unity.
2. Descarga el último archivo `Aptos.Unity.unitypackage` desde el [Unity Asset Store](https://assetstore.unity.com/packages/decentralization/aptos-sdk-244713).
3. Haz clic en **Assets** → **Import Packages** → **Custom Package** y selecciona el archivo descargado.

### Instalar por Unity Package Manager

1. Abre la ventana del [Unity Package Manager](https://docs.unity3d.com/Manual/upm-ui.html).
2. Haz clic en el botón agregar **+** en la barra de estado superior.
3. Selecciona _Add package from git URL_ del menú desplegable.
4. Ingresa la URL _[https://github.com/aptos-labs/Aptos-Unity-SDK.git](https://github.com/aptos-labs/Aptos-Unity-SDK.git)_ y haz clic en **Add**.
