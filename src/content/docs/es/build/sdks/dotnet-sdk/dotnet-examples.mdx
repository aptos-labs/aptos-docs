El SDK de .NET de Aptos proporciona varios ejemplos para ayudarte a comenzar con el SDK. Puedes encontrar los ejemplos en el
repositorio [aptos-labs/aptos-dotnet-sdk](https://github.com/aptos-labs/aptos-dotnet-sdk).

<br />

<LinkCard href="https://github.com/aptos-labs/aptos-dotnet-sdk/tree/main/Aptos.Examples" title="Ejemplos del SDK de .NET de Aptos" description="Aplicaciones de ejemplo para el SDK de .NET." target="_blank" />

<Steps>
  1. Instalar .NET

     Para ejecutar los ejemplos, necesitarás instalar el SDK de .NET. Puedes descargar el SDK de .NET desde el
     sitio web [dotnet.microsoft.com](https://dotnet.microsoft.com/en-us/download).

  2. Clonar el Repositorio

     Clona el repositorio ejecutando el siguiente comando:

     ```shellscript
     git clone https://github.com/aptos-labs/aptos-dotnet-sdk.git
     ```

  3. Ejecutar los Ejemplos

     Puedes ejecutar los ejemplos navegando al directorio `Aptos.Examples` y ejecutando el
     comando `dotnet run --framework net8.0`.

     ```shellscript
     cd Aptos.Examples
     dotnet run --framework net8.0
     ```

  4. Seleccionar un Ejemplo

     Al ejecutar los ejemplos, se te pedirá que selecciones un ejemplo. Puedes seleccionar el ejemplo
     ingresando el número del ejemplo que quieres ejecutar o navegando con las teclas de flecha.

     ![examples-demonstration](https://i.imgur.com/YS140Zb.png)
</Steps>
