Esta guía te llevará a través del proceso de integrar el SDK .NET de Aptos.

<Steps>
  1. Instalar el SDK Unity de Aptos

     #### Opción 1: Importar vía Unity Package Manager (UPM)

     1. Abre la ventana Package Manager (Window | Package Manager)
     2. Haz clic en el botón + en la esquina superior izquierda de la ventana, y selecciona _Add package from git URL..._
     3. Ingresa la siguiente URL y haz clic en el botón Add

     ```shellscript
     https://github.com/aptos-labs/unity-sdk.git?path=/Packages/com.aptoslabs.aptos-unity-sdk
     ```

     #### Opción 2: Importar vía `unitypackage`

     1. Ve a los [`aptos-labs/unity-sdk Releases`](https://github.com/aptos-labs/unity-sdk/releases) y descarga el último release.
     2. Arrastra y suelta el archivo `.unitypackage` en tu proyecto Unity.

  2. Usar el SDK de Aptos

     Importa el namespace `Aptos` en tu script C# y usa el SDK.

     ```csharp {2-2,8-8,11-15}
     using UnityEngine;
     using Aptos;

     class Example : MonoBehaviour
     {
         public void Start()
         {
             PrintLedgerInfo();
         }

         async void PrintLedgerInfo() {
             var client = new AptosUnityClient(Networks.Mainnet);
             var ledgerInfo = await client.Block.GetLedgerInfo();
             Debug.Log(ledgerInfo.BlockHeight);
         }

     }
     ```
</Steps>
