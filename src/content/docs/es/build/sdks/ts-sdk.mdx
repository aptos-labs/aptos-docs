El SDK de TypeScript te permite conectar, explorar e interactuar en la blockchain de Aptos. ¡Puedes usarlo para solicitar datos, enviar transacciones, configurar entornos de prueba y más!

```shellscript npm2yarn
npm i @aptos-labs/ts-sdk
```
