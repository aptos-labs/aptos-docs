## ¿Qué es LLMs.txt?

Soportamos archivos [LLMs.txt](https://llmstxt.org/) para hacer que la documentación de Aptos esté disponible para modelos de lenguaje grandes (LLMs). Esta característica ayuda a las herramientas de IA a entender mejor la blockchain de Aptos, su lenguaje Move, SDKs y patrones de desarrollo.

## Rutas Disponibles

Proporcionamos las siguientes rutas de LLMs.txt para ayudar a las herramientas de IA a acceder a nuestra documentación:

- [llms.txt](/llms.txt) - Contiene una visión general estructurada de los archivos LLMs.txt de Aptos
- [llms-small.txt](/llms-small.txt) - Proporciona una versión condensada de la documentación, optimizada para ventanas de contexto más pequeñas
- [llms-full.txt](/llms-full.txt) - Proporciona la documentación completa y exhaustiva para todos los conceptos de Aptos

## Uso con Herramientas de IA

### Cursor

Usa la función `@Docs` en Cursor para incluir los archivos LLMs.txt en tu proyecto. Esto ayuda a Cursor a proporcionar sugerencias de código y documentación más precisas para el desarrollo en Aptos.

[Leer más sobre @Docs en Cursor](https://docs.cursor.com/context/@-symbols/@-docs)

### GitHub Copilot

GitHub Copilot puede aprovechar la información en estos archivos LLMs.txt para proporcionar mejor asistencia al desarrollar aplicaciones en Aptos. Puedes referenciar estos archivos en GitHub Copilot Chat usando las siguientes URLs:

```
https://aptos.dev/llms.txt
https://aptos.dev/llms-small.txt
https://aptos.dev/llms-full.txt
```

### Windsurf

Referencia los archivos LLMs.txt usando `@` o en tus archivos `.windsurfrules` para mejorar la comprensión de Windsurf sobre el desarrollo en Aptos.

[Leer más sobre Windsurf Memories](https://docs.codeium.com/windsurf/memories#memories-and-rules)

### Otras Herramientas de IA

Cualquier herramienta de IA que soporte LLMs.txt puede usar estas rutas para entender mejor Aptos. Simplemente apunta tu herramienta a cualquiera de las rutas mencionadas arriba.
