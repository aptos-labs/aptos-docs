## Que es LLMs.txt?

Compatibilizamos archivos [LLMs.txt](https://llmstxt.org/) para que la documentacion de Aptos sea mas facil de consumir por modelos de lenguaje grandes (LLMs).
Esto ayuda a las herramientas de IA a entender mejor Aptos, el lenguaje Move, los SDKs y los patrones de desarrollo.

## Rutas disponibles

Ofrecemos las siguientes rutas de LLMs.txt para ayudar a las herramientas de IA a acceder a nuestra documentacion:

- [llms.txt](/llms.txt) - Un archivo de enrutamiento compacto que apunta a la documentacion de Aptos mas util y a las exportaciones legibles por maquinas
- [llms-small.txt](/llms-small.txt) - Un subconjunto curado y de bajo consumo de tokens para asistentes de IDE y ventanas de contexto mas pequenas
- [llms-full.txt](/llms-full.txt) - El corpus completo de la documentacion de Aptos en Markdown renderizado para contextos amplios

**URL conocida (well-known):** Algunas herramientas buscan [`https://aptos.dev/.well-known/llms.txt`](https://aptos.dev/.well-known/llms.txt). Esa URL redirige de forma permanente a [`/llms.txt`](/llms.txt) (mismo contenido).

## Acceso Markdown por pagina

Cada pagina de documentacion tambien esta disponible como Markdown renderizado agregando `.md` al final de la URL:

| URL de la pagina                                      | URL Markdown                                             |
| ----------------------------------------------------- | -------------------------------------------------------- |
| `https://aptos.dev/build/guides/first-transaction`    | `https://aptos.dev/build/guides/first-transaction.md`    |
| `https://aptos.dev/build/sdks/ts-sdk`                 | `https://aptos.dev/build/sdks/ts-sdk.md`                 |
| `https://aptos.dev/zh/build/guides/first-transaction` | `https://aptos.dev/zh/build/guides/first-transaction.md` |

Esto es util para herramientas de IA que necesitan leer paginas individuales con pocos tokens en lugar de ingerir toda la documentacion. [llms.txt](/llms.txt) funciona como un enrutador compacto, mientras que [llms-small.txt](/llms-small.txt) y [llms-full.txt](/llms-full.txt) ofrecen exportaciones curadas y completas del corpus.

## Uso con herramientas de IA

### Claude Code

Agrega contexto de la documentacion de Aptos a proyectos de [Claude Code](https://docs.anthropic.com/en/docs/agents-and-tools/claude-code/overview):

1. Agrega `https://aptos.dev/llms.txt` al archivo `CLAUDE.md` de tu proyecto como referencia
2. Claude Code leera el indice y obtendra las paginas relevantes mediante URLs `.md` cuando sea necesario
3. Para una integracion mas profunda con herramientas on-chain, consulta [Aptos MCP for Claude Code](/build/ai/aptos-mcp/claude)

### Cursor

Usa la funcion `@Docs` de Cursor para incluir los archivos LLMs.txt en tu proyecto. Esto ayuda a Cursor a ofrecer sugerencias de codigo y documentacion mas precisas para Aptos.

[Lee mas sobre @Docs en Cursor](https://docs.cursor.com/context/@-symbols/@-docs)

### GitHub Copilot

GitHub Copilot puede aprovechar la informacion de estos archivos LLMs.txt para ofrecer mejor asistencia al desarrollar aplicaciones sobre Aptos. Puedes hacer referencia a estos archivos en GitHub Copilot Chat usando las siguientes URLs:

```
https://aptos.dev/llms.txt
https://aptos.dev/llms-small.txt
https://aptos.dev/llms-full.txt
```

### Windsurf

Haz referencia a los archivos LLMs.txt mediante `@` o dentro de tus archivos `.windsurfrules` para mejorar la comprension de Windsurf sobre el desarrollo en Aptos.

[Lee mas sobre Windsurf Memories](https://docs.codeium.com/windsurf/memories#memories-and-rules)

### Claude.ai

Agrega la documentacion de Aptos como fuente de conocimiento en [Claude Projects](https://claude.ai):

1. Abre [Claude.ai](https://claude.ai) y crea o abre un proyecto
2. Haz clic en **Add content**
3. Pega `https://aptos.dev/llms-full.txt` como fuente por URL
4. Claude tendra contexto de la documentacion de Aptos en todas las conversaciones dentro de ese proyecto

### ChatGPT

Haz referencia a la documentacion de Aptos directamente en conversaciones de ChatGPT:

- Pega `https://aptos.dev/llms-full.txt` en tu mensaje y pide a ChatGPT que lo lea antes de responder preguntas sobre Aptos
- Para [GPTs personalizados](https://openai.com/index/introducing-gpts/): agrega `https://aptos.dev/llms.txt` como fuente de conocimiento en GPT Builder

### Otras herramientas de IA

Cualquier herramienta de IA compatible con LLMs.txt puede usar estas rutas para entender mejor Aptos. Solo apunta tu herramienta a cualquiera de las rutas anteriores.
