Aptos Testnet Faucet te permite solicitar al instante **APT** gratuitos en la **testnet de Aptos**.\
Estos tokens son **solo para fines de prueba** — no tienen valor en el mundo real y no se pueden intercambiar por APT de mainnet.

Usa el faucet para:

- Desplegar y probar **contratos inteligentes Move**
- Experimentar con los SDKs de Aptos (TypeScript, Python, Rust, y más)
- Construir y perfeccionar dApps antes de lanzarlas a mainnet

## Cómo usar el Faucet

1. **Inicia sesión** con tu cuenta de Google para prevenir abusos.
2. **Introduce tu dirección de billetera de testnet de Aptos**.
3. Haz clic en **Solicitar APT** para recibir los tokens al instante.
4. Verifica el saldo en tu billetera — los tokens deberían aparecer en pocos segundos.

> **Consejo:** Si no tienes una cuenta de testnet, puedes [crear una cuenta con el SDK de TypeScript de Aptos](/es/build/sdks/ts-sdk/quickstart) (método para desarrolladores) o [instalar una billetera de Aptos y cambiar a testnet](https://aptosfoundation.org/ecosystem/projects/wallets) (método con interfaz gráfica).
