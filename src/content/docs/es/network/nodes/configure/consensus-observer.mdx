El observador de consenso es una nueva técnica de diseminación de datos que reduce el tiempo de sincronización de bloques, y
latencias de transacciones para fullnodes de Aptos (es decir, VFNs y PFNs). Se ha demostrado que reduce las latencias de transacciones
de extremo a extremo en Mainnet en 10% a 50%, dependiendo de la carga (es decir, transacciones por segundo).

Más información sobre el observador de consenso se puede encontrar en el AIP: [AIP-93: Observador de Consenso](https://github.com/aptos-foundation/AIPs/blob/main/aips/aip-93.md).

<Aside type="tip">
  **Public Fullnodes (PFNs)**<br />
  Las configuraciones descritas en este documento son solo para public fullnodes (PFNs).
  El observador de consenso ya ha sido habilitado (por defecto) para validators y validator fullnodes (VFNs).
  No se recomienda cambiar las configuraciones del observador de consenso para validators y VFNs.
</Aside>

## Configurar observador de consenso

<Aside type="caution" emoji="❗">
  **Requisitos Mínimos de Hardware**<br />
  El observador de consenso no está habilitado en PFNs (por defecto) porque requiere que los nodos cumplan los
  [requisitos mínimos de hardware](/network/nodes/full-node/pfn-requirements#hardware-requirements).
  Si tu nodo está sub-provisionado y no cumple estos requisitos, habilitar el observador de consenso
  resultará en rendimiento degradado porque requiere más CPU que la sincronización de estado tradicional. Solo
  habilita el observador de consenso si tu nodo cumple los requisitos mínimos de hardware.
</Aside>

### Habilitar observador de consenso

Para habilitar el observador de consenso en tu PFN, agrega lo siguiente a tu archivo de configuración de nodo:

```yaml filename="fullnode.yaml"
consensus:
  enable_pre_commit: false

consensus_observer:
  observer_enabled: true
  publisher_enabled: true
```

### Deshabilitar observador de consenso

Para deshabilitar el observador de consenso en tu PFN, agrega lo siguiente a tu archivo de configuración de nodo:

```yaml filename="fullnode.yaml"
consensus:
  enable_pre_commit: true

consensus_observer:
  observer_enabled: false
  publisher_enabled: false
```
