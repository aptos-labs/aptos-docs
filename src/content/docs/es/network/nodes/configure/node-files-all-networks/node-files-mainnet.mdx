Cuando despliegues un nodo Aptos en **mainnet**, podrías necesitar descargar los archivos listados en esta página. Los archivos están
organizados por tipo de nodo.

<Aside type="note">
  **Acceso a archivos**<br />
  Dependiendo del tipo de nodo que estés desplegando, y el método de despliegue, podrías necesitar descargar algunos
  o todos los archivos listados abajo.
</Aside>

## Archivos de Validator

Todos los archivos listados abajo son para nodos validator.

### docker-compose.yaml

- **Repositorio Git:** `aptos-core`
- **Rama Git:** `mainnet` en [https://github.com/aptos-labs/aptos-core](https://github.com/aptos-labs/aptos-core)
- **Comando para descargar:**
  ```shellscript filename="Terminal"
  wget -O docker-compose.yaml https://raw.githubusercontent.com/aptos-labs/aptos-core/mainnet/docker/compose/aptos-node/docker-compose.yaml
  ```
