Antes de que puedas ejecutar un validator y validator fullnode (VFN) en una red de Aptos, necesitarás seleccionar un método de despliegue
para tus nodos. Las guías abajo proporcionan instrucciones paso a paso para desplegar un nodo validator y VFN
en varias plataformas, y a través de diferentes redes de Aptos.

Una vez que tus nodos estén desplegados, puedes conectarlos a una red de Aptos inicializando tu pool de staking y uniéndote
al conjunto de validators. Ver [Conectar Nodos](/es/network/nodes/validator-node/connect-nodes) para más información.

<Aside type="caution">
  Antes de seleccionar un método de despliegue, asegúrate de leer y entender los [Requisitos del Nodo](/es/network/nodes/validator-node/node-requirements) primero.
</Aside>
