title: "Últimas Versiones de Aptos"
---

Cada versión de Aptos se puede encontrar en el sitio de GitHub para [versiones de Aptos-core](https://github.com/aptos-labs/aptos-core/releases).
Cada versión está reflejada por las siguientes ramas de git:

- [Última Versión de Mainnet](https://github.com/aptos-labs/aptos-core/tree/mainnet)
- [Última Versión de Testnet](https://github.com/aptos-labs/aptos-core/tree/testnet)
- [Última Versión de Devnet](https://github.com/aptos-labs/aptos-core/tree/devnet)

Aptos típicamente realiza múltiples versiones de devnet para cada versión de testnet y mainnet.
Por lo tanto, las versiones de devnet usan hashes de commit para el seguimiento en lugar de
números de versión. Las versiones de testnet y mainnet usualmente tienen una correlación
uno a uno, lo que significa que cada versión de testnet se convierte en mainnet.

Los hot fixes son excepciones que pueden ocurrir en mainnet para abordar problemas urgentes en
producción. Consulta el [Proceso de Versión de Aptos](https://github.com/aptos-labs/aptos-core/blob/main/RELEASE.md)
para más detalles.

## Actualizar nodos

Si eres un operador de nodo, [actualiza tus nodos con la nueva versión](/network/nodes/full-node/modify/update-fullnode-with-new-releases).

## Suscribirse a Anuncios de Versión

### Suscribirse vía GitHub

1. Ve al repositorio [aptos-labs/aptos-core](https://github.com/aptos-labs/aptos-core).
2. Abre el menú **Watch** y selecciona **Custom**.
3. Selecciona la casilla **Releases** y haz clic en **Apply**.

### Suscribirse vía [Aptos Discord](https://discord.gg/aptosnetwork)

Únete al servidor de Discord de Aptos para interactuar con nosotros y nuestra comunidad. También publicamos próximas versiones en estos canales.

- [#mainnet-release](https://discord.com/channels/945856774056083548/1042502400507916349)
- [#testnet-release](https://discord.com/channels/945856774056083548/1025614160555413545)
- [#devnet-release](https://discord.com/channels/945856774056083548/956692649430093904)

### Suscribirse vía [Aptos Twitter](https://x.com/AptosRelease)

Sigue [@AptosRelease](https://x.com/AptosRelease) en Twitter para obtener las últimas actualizaciones sobre nuestras próximas
versiones de mainnet y ser notificado cuando sea momento de actualizar tu nodo. Cada
pocos días, [@AptosRelease](https://x.com/AptosRelease) twitteará una cuenta regresiva para recordarte que actualices a
la última versión. _Nota: ¡No publicamos sobre hotfixes aquí!_

## Proceso de Versión de Aptos

Para entender cómo realizamos las versiones, revisa el [Proceso de Versión de Aptos](https://github.com/aptos-labs/aptos-core/blob/main/RELEASE.md).
