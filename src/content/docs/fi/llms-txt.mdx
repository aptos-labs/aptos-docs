## Mikä on LLMs.txt?

Tuemme [LLMs.txt](https://llmstxt.org/)-tiedostoja tehdäksemme Aptos-dokumentaation saataville suurille kielimalleille (LLMs). Tämä ominaisuus auttaa AI-työkaluja ymmärtämään paremmin Aptos-blokiketjun, Move-kielen, SDK:t ja kehitysmallit.

## Käytettävissä olevat reitit

Tarjoamme seuraavat LLMs.txt-reitit auttaaksemme AI-työkaluja pääsemään käsiksi dokumentaatioomme:

- [llms.txt](/llms.txt) - Sisältää strukturoidun yleiskatsauksen Aptos LLMs.txt-tiedostoista
- [llms-small.txt](/llms-small.txt) - Tarjoaa tiivistetyn version dokumentaatiosta, optimoituna pienemmille konteksti-ikkunoille
- [llms-full.txt](/llms-full.txt) - Tarjoaa täydellisen ja kattavan dokumentaation kaikille Aptos-konsepteille

## Käyttö AI-työkalujen kanssa

### Cursor

Käytä `@Docs`-ominaisuutta Cursorissa sisällyttääksesi LLMs.txt-tiedostot projektiisi. Tämä auttaa Cursoria tarjoamaan tarkempia koodiehdotuksia ja dokumentaatiota Aptos-kehitykselle.

[Lue lisää @Docsista Cursorissa](https://docs.cursor.com/context/@-symbols/@-docs)

### GitHub Copilot

GitHub Copilot voi hyödyntää näiden LLMs.txt-tiedostojen tietoja tarjotakseen paremman avun Aptos-sovellusten kehittämisessä. Voit viitata näihin tiedostoihin GitHub Copilot Chatissa käyttämällä seuraavia URL-osoitteita:

```
https://aptos.dev/llms.txt
https://aptos.dev/llms-small.txt
https://aptos.dev/llms-full.txt
```

### Windsurf

Viittaa LLMs.txt-tiedostoihin käyttämällä `@` tai `.windsurfrules`-tiedostoissasi parantaaksesi Windsurfin ymmärrystä Aptos-kehityksestä.

[Lue lisää Windsurf Memoriesista](https://docs.codeium.com/windsurf/memories#memories-and-rules)

### Muut AI-työkalut

Mikä tahansa AI-työkalu, joka tukee LLMs.txt:tä, voi käyttää näitä reittejä ymmärtääkseen Aptosin paremmin. Ohjaa työkalusi vain johonkin yllä olevista reiteistä.
