Aptos Testnet Hana mahdollistaa kehittäjille välittömän pyynnön ilmaisten APT-tokenien saamisesta **Aptos testnetissä**. Nämä tokenit on tarkoitettu **vain testauskäyttöön** - niillä ei ole todellista arvoa eikä niitä voida vaihtaa mainnet APT:hen.

Käytä hanua:

- **Move Älykkäiden Sopimusten** käyttöönotto ja testaus
- Aptos SDK:iden kokeilu (TypeScript, Python, Rust ja enemmän)
- dAppien rakentaminen ja parantaminen ennen mainnet-lanseerausta

<Faucet />

## Hanan käyttö

1. **Kirjaudu sisään** Google-tililläsi.
2. **Syötä Aptos testnet lompakkosi osoite**.
3. Klikkaa **Pyydä APT** saadaksesi tokeneita välittömästi.
4. Tarkista lompakkosi saldo — tokenit ilmestyvät muutamassa sekunnissa.

> **Vinkki:** Jos sinulla ei ole testnet-tiliä, voit joko [Luoda tilin Aptos TypeScript SDK:lla](/build/sdks/ts-sdk/quickstart) (kehittäjämenetelmä) tai [Asentaa Aptos-lompakon ja vaihtaa Testnetiin](https://aptosfoundation.org/ecosystem/projects/wallets) (UI-menetelmä) aloittaaksesi.
