Aptos-verkko koostuu kolmesta solmutyypistä: validoija-solmut, validoija-täyssolmut (VFN) ja julkiset täyssolmut (PFN). Osallistuaksesi konsensukseen sinun täytyy ajaa validoija-solmu ja tehdä staking vähimmäismäärästä apukolikoita. VFN:t ja PFN:t eivät ole pakollisia konsensusosallistumisen kannalta, mutta ne ovat välttämättömiä jakamaan blokiketjutietoja ja mahdollistamaan ekosysteemipalvelut, kuten indeksointi, kyselyt ja RESTful API -palvelut ([Aptos APIs](/build/apis) katso). VFN:t voivat ajaa vain validoijaoperaattorit, kun taas PFN:n voi ajaa kuka tahansa. Lue lisää eri solmutyypeistä kohdasta [Blokiketjun syväsukellus](/network/blockchain).

Tämä osio tarjoaa yksityiskohtaiset vaihe vaiheelta -ohjeet Aptos-solmujen käyttöönottoon ja hallintaan eri ympäristöissä. Se kuvaa myös kaikki mitä tarvitset stakingiin ja konsensus- sekä hallinto-osallistumiseen.
