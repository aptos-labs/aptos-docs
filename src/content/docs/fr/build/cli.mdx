L'interface de ligne de commande (CLI) Aptos est un outil pour vous aider à compiler et tester les contrats Move. Elle peut également vous aider à jouer rapidement avec les fonctionnalités Aptos sur la chaîne.

Pour les utilisateurs plus avancés, la CLI peut également être utilisée pour exécuter un réseau Aptos privé (pour aider à tester le code localement) et peut être utile pour gérer un nœud de réseau.
