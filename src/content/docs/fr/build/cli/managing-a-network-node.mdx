Si vous exécutez un [nœud validateur ou nœud complet validateur (VFN)](/network/nodes/validator-node), vous pouvez utiliser la CLI pour interagir avec votre nœud.

Plus précisément, vous pouvez utiliser la CLI pour :

1. [Gérer les pools de jalonnement que vous possédez](/network/nodes/validator-node/connect-nodes/staking-pool-operations).
2. [Voter sur les propositions](/network/nodes/validator-node/connect-nodes/staking-pool-voter).

Au-delà de cela, vous pouvez exécuter cette commande d'aide pour voir plus de commandes spécialisées que la CLI peut effectuer concernant l'exploitation de votre nœud :

```shellscript filename="Terminal"
aptos node --help
```
