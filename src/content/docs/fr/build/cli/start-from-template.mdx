Suivez les étapes ci-dessous pour commencer rapidement.

<Steps>
  1. Initialiser

     Exécutez ce qui suit pour initialiser un package en utilisant le modèle `hello-blockchain` :

     ```shellscript filename="Terminal"
     aptos move init --name hello_blockchain --template hello-blockchain
     ```

  2. Commencer à construire

     Le modèle crée un fichier `hello_blockchain.move` sous `sources` pour vous aider à commencer.

     <RemoteCodeblock permalink="https://github.com/aptos-labs/aptos-core/blob/afd3706c17bcccfb39a9d6059aecbfa648ed295d/aptos-move/move-examples/hello_blockchain/sources/hello_blockchain.move#L1-L64" />

  3. Voir tous les modèles

     Exécutez la commande suivante pour voir tous les modèles (et pour une aide générale sur l'initialisation d'un package) :

     ```shellscript
     aptos move init --help
     ```
</Steps>
