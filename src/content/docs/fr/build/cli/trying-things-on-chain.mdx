La CLI peut être un outil pratique pour rechercher rapidement des données sur la chaîne et envoyer des transactions depuis vos comptes.

La façon la plus courante de spécifier avec quels comptes vous voulez interagir est à travers les profils. Vous pouvez créer un nouveau profil sur la CLI en exécutant la commande suivante :

```shellscript filename="Terminal"
aptos init --profile <your-profile-name>
```

Si une commande prend un compte, vous pouvez passer le nom d'un profil à la place. Si une commande utilise implicitement le profil par défaut, elle aura généralement un paramètre optionnel pour utiliser un profil spécifié à la place que vous pouvez trouver en exécutant `aptos <your-command> --help`.
