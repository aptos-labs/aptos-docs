  label: "Solana → Aptos"

Pour en savoir plus sur les différences et similarités, consultez [Aptos Learn](https://learn.aptoslabs.com/en/tutorials/solana-to-aptos-guide/cheat-sheet?workshop=solana-to-aptos)

|                            | Solana                                                                                                             | Aptos                                                                 |
|----------------------------|--------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------|
| **Contrats Intelligents**  | Rust, SVM                                                                                                          | Move, MoveVM                                                          |
| **Frais de Transaction**   | Faibles                                                                                                            | Faibles                                                               |
| **Parallélisation**        | Parallélisation pessimiste, besoin de déclarer tous les comptes d'écriture                                         | Parallélisation optimiste, la chaîne déduit les comptes d'écriture pour vous |
| **Support de Compte de Contrat** | Compte PDA                                                                                                         | [Objet](/build/smart-contracts/objects) ou [compte ressource](/build/smart-contracts/resource-accounts) (encouragez l'utilisation d'objet à la place) |
| **Stockage de Données**    | Données stockées dans le compte appartenant aux programmes                                                        | Données stockées comme ressource sous le compte utilisateur ou objet |
| **Niveau de Stockage**     | Niveau programme                                                                                                   | Global quand stocké sous objet                                        |
| **Mentalité de Stockage**  | Données utilisateur stockées de manière distribuée sous compte                                                     | Données utilisateur stockées de manière distribuée sous objet         |
| **Exemple de Code**        | [Contrat de liste de tâches sur Solana](https://github.com/aptos-labs/move-by-examples/tree/main/advanced-todo-list/solana) | [Contrat de liste de tâches sur Aptos](https://github.com/aptos-labs/move-by-examples/tree/main/advanced-todo-list/aptos) |
| **ID de l'Appelant**       | `signer`                                                                                                           | `signer`                                                              |
| **Mise à Niveau**          | Le programme est évolutif                                                                                          | Le module est évolutif                                                |
| **Type de Dispatch**       | Dispatch statique                                                                                                  | Dispatch statique                                                     |
| **Standards FT**           | Programme de token                                                                                                 | [Coin](/build/smart-contracts/aptos-coin) (legacy) et [Standard d'Actif Fongible](/build/smart-contracts/fungible-asset) |
| **Standards NFT**          | Programme de token                                                                                                 | [Standard d'Actif Numérique](/build/smart-contracts/digital-asset)    |
| **Interaction Blockchain** | Bibliothèque Solana web3.js                                                                                        | [SDK TypeScript Aptos](/build/sdks/ts-sdk)                            |
