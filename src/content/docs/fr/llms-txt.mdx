## Qu'est-ce que LLMs.txt ?

Nous prenons en charge les fichiers [LLMs.txt](https://llmstxt.org/) pour rendre la documentation Aptos disponible aux grands modèles de langage
(LLMs). Cette fonctionnalité aide les outils d'IA à mieux comprendre la blockchain Aptos, son langage Move, les SDK et les modèles de développement.

## Routes disponibles

Nous fournissons les routes LLMs.txt suivantes pour aider les outils d'IA à accéder à notre documentation :

- [llms.txt](/llms.txt) - Contient un aperçu structuré des fichiers Aptos LLMs.txt
- [llms-small.txt](/llms-small.txt) - Fournit une version condensée de la documentation, optimisée pour les fenêtres de contexte plus petites
- [llms-full.txt](/llms-full.txt) - Fournit la documentation complète et exhaustive pour tous les concepts Aptos

## Utilisation avec les outils d'IA

### Cursor

Utilisez la fonctionnalité `@Docs` dans Cursor pour inclure les fichiers LLMs.txt dans votre projet. Cela aide Cursor à fournir des suggestions
de code et une documentation plus précises pour le développement Aptos.

[En savoir plus sur @Docs dans Cursor](https://docs.cursor.com/context/@-symbols/@-docs)

### GitHub Copilot

GitHub Copilot peut exploiter les informations contenues dans ces fichiers LLMs.txt pour fournir une meilleure assistance lors du développement d'applications Aptos. Vous pouvez référencer ces fichiers dans votre GitHub Copilot Chat en utilisant les URL suivantes :

```
https://aptos.dev/llms.txt
https://aptos.dev/llms-small.txt
https://aptos.dev/llms-full.txt
```

### Windsurf

Référencez les fichiers LLMs.txt en utilisant `@` ou dans vos fichiers `.windsurfrules` pour améliorer la compréhension de Windsurf du développement Aptos.

[En savoir plus sur Windsurf Memories](https://docs.codeium.com/windsurf/memories#memories-and-rules)

### Autres outils d'IA

Tout outil d'IA qui prend en charge LLMs.txt peut utiliser ces routes pour mieux comprendre Aptos. Pointez simplement votre outil vers l'une des
routes ci-dessus.
