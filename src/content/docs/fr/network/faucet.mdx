Le Robinet Aptos Testnet permet aux développeurs de demander instantanément des tokens APT gratuits sur le **réseau testnet Aptos**. Ces tokens sont uniquement pour **des fins de test** — ils n'ont aucune valeur réelle et ne peuvent pas être échangés contre des APT mainnet.

Utilisez le robinet pour :

- Déployer et tester des **contrats intelligents Move**
- Expérimenter avec les SDKs Aptos (TypeScript, Python, Rust, et plus)
- Construire et affiner des dApps avant de les lancer sur le mainnet

<Faucet />

## Comment Utiliser le Robinet

1. **Connectez-vous** avec votre compte Google.
2. **Entrez votre adresse de portefeuille testnet Aptos**.
3. Cliquez sur **Demander des APT** pour recevoir instantanément des tokens.
4. Vérifiez le solde de votre portefeuille — les tokens devraient apparaître en quelques secondes.

> **Astuce :** Si vous n'avez pas de compte testnet, vous pouvez soit [Créer un Compte avec le SDK TypeScript Aptos](/build/sdks/ts-sdk/quickstart) (méthode développeur) ou [Installer un Portefeuille Aptos et Basculer vers Testnet](https://aptosfoundation.org/ecosystem/projects/wallets) (méthode UI) pour commencer.
