Le réseau Aptos se compose de trois types de nœuds : les nœuds validateurs, les nœuds complets validateurs (VFNs) et les nœuds complets publics (PFNs). Pour participer au consensus, vous devez exécuter un nœud validateur et miser le montant minimum de pièces utilitaires. Les VFNs et PFNs ne sont pas requis pour participer au consensus, mais ils sont nécessaires pour distribuer les données de blockchain et permettre les services d'écosystème, par exemple, l'indexation, l'interrogation et les services API RESTful (voir [APIs Aptos](/build/apis)). Les VFNs ne peuvent être exécutés que par les opérateurs de validateurs, tandis que les PFNs peuvent être exécutés par n'importe qui. Vous pouvez en apprendre plus sur les différents types de nœuds dans la section [Plongée Profonde dans la Blockchain](/network/blockchain).

Cette section fournit des instructions détaillées, étape par étape, sur comment déployer et opérer les nœuds Aptos dans différents environnements. Elle décrit également tout ce dont vous avez besoin pour miser et participer au consensus et à la gouvernance.
