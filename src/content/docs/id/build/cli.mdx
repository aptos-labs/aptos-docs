Command line interface (CLI) Aptos adalah alat untuk membantu Anda mengkompilasi dan menguji kontrak Move. Ini juga dapat membantu Anda dengan cepat bermain dengan fitur Aptos on-chain.

Untuk pengguna yang lebih advanced, CLI juga dapat digunakan untuk menjalankan jaringan Aptos privat (untuk membantu menguji kode secara lokal) dan dapat membantu mengelola node jaringan.
