## Apa itu LLMs.txt?

Kami mendukung file [LLMs.txt](https://llmstxt.org/) untuk membuat dokumentasi Aptos tersedia untuk model bahasa besar
(LLM). Fitur ini membantu alat AI memahami blockchain Aptos, bahasa Move, SDK, dan pola pengembangan dengan lebih baik.

## Rute yang Tersedia

Kami menyediakan rute LLMs.txt berikut untuk membantu alat AI mengakses dokumentasi kami:

- [llms.txt](/llms.txt) - Berisi ikhtisar terstruktur file LLMs.txt Aptos
- [llms-small.txt](/llms-small.txt) - Menyediakan versi ringkas dokumentasi, dioptimalkan untuk jendela konteks yang lebih kecil
- [llms-full.txt](/llms-full.txt) - Menyediakan dokumentasi lengkap dan komprehensif untuk semua konsep Aptos

## Penggunaan dengan Alat AI

### Cursor

Gunakan fitur `@Docs` di Cursor untuk menyertakan file LLMs.txt dalam proyek Anda. Ini membantu Cursor memberikan saran
kode dan dokumentasi yang lebih akurat untuk pengembangan Aptos.

[Baca lebih lanjut tentang @Docs di Cursor](https://docs.cursor.com/context/@-symbols/@-docs)

### GitHub Copilot

GitHub Copilot dapat memanfaatkan informasi dalam file LLMs.txt ini untuk memberikan bantuan yang lebih baik saat mengembangkan aplikasi Aptos. Anda dapat mereferensikan file-file ini di GitHub Copilot Chat Anda dengan menggunakan URL berikut:

```
https://aptos.dev/llms.txt
https://aptos.dev/llms-small.txt
https://aptos.dev/llms-full.txt
```

### Windsurf

Referensikan file LLMs.txt menggunakan `@` atau dalam file `.windsurfrules` Anda untuk meningkatkan pemahaman Windsurf tentang pengembangan Aptos.

[Baca lebih lanjut tentang Windsurf Memories](https://docs.codeium.com/windsurf/memories#memories-and-rules)

### Alat AI Lainnya

Alat AI apa pun yang mendukung LLMs.txt dapat menggunakan rute ini untuk memahami Aptos dengan lebih baik. Cukup arahkan alat Anda ke salah satu
rute di atas.
