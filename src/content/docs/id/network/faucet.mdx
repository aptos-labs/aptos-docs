Faucet Testnet Aptos memungkinkan pengembang untuk meminta token APT gratis secara instan di **jaringan testnet Aptos**. Token ini hanya untuk **tujuan pengujian** - mereka tidak memiliki nilai nyata dan tidak dapat ditukar dengan APT mainnet.

Gunakan faucet untuk:

- Menyebarkan dan menguji **kontrak pintar Move**
- Bereksperimen dengan SDK Aptos (TypeScript, Python, Rust, dan lainnya)
- Membangun dan menyempurnakan dApp sebelum meluncurkan ke mainnet

<Faucet />

## Cara Menggunakan Faucet

1. **Masuk** dengan akun Google Anda.
2. **Masukkan alamat dompet testnet Aptos Anda**.
3. Klik **Minta APT** untuk menerima token secara instan.
4. Periksa saldo dompet Anda — token akan muncul dalam beberapa detik.

> **Tips:** Jika Anda tidak memiliki akun testnet, Anda dapat [Membuat Akun dengan Aptos TypeScript SDK](/build/sdks/ts-sdk/quickstart) (metode pengembang) atau [Menginstal Dompet Aptos dan Beralih ke Testnet](https://aptosfoundation.org/ecosystem/projects/wallets) (metode UI) untuk memulai.
