Jaringan Aptos terdiri dari tiga jenis node: node validator, node lengkap validator (VFN), dan node lengkap publik (PFN). Untuk berpartisipasi dalam konsensus, Anda harus menjalankan node validator dan melakukan staking jumlah minimum koin utilitas. VFN dan PFN tidak diperlukan untuk berpartisipasi dalam konsensus, tetapi mereka diperlukan untuk mendistribusikan data blockchain dan memungkinkan layanan ekosistem seperti pengindeksan, kueri, dan layanan API RESTful (lihat [Aptos APIs](/build/apis)). VFN hanya dapat dijalankan oleh operator validator, sedangkan PFN dapat dijalankan oleh siapa saja. Anda dapat mempelajari lebih lanjut tentang berbagai jenis node di bagian [Mergulap Dalam Blockchain](/network/blockchain).

Bagian ini memberikan petunjuk langkah demi langkah yang detail untuk menyebarkan dan mengoperasikan node Aptos di berbagai lingkungan. Ini juga menjelaskan semua yang Anda perlukan untuk melakukan staking dan berpartisipasi dalam konsensus dan tata kelola.
