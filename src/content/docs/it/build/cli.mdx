L'interfaccia a riga di comando (CLI) di Aptos è uno strumento per aiutarti a compilare e testare contratti Move. Può anche aiutarti a sperimentare rapidamente con le funzionalità di Aptos on-chain.

Per utenti più avanzati, la CLI può anche essere usata per eseguire una rete Aptos privata (per aiutare a testare il codice localmente) e può essere utile per gestire un nodo di rete.
