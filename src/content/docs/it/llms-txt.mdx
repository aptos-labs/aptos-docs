## Cos'è LLMs.txt?

Supportiamo i file [LLMs.txt](https://llmstxt.org/) per rendere la documentazione di Aptos disponibile ai grandi modelli linguistici
(LLM). Questa funzionalità aiuta gli strumenti di IA a comprendere meglio la blockchain Aptos, il suo linguaggio Move, gli SDK e i pattern di sviluppo.

## Route Disponibili

Forniamo le seguenti route LLMs.txt per aiutare gli strumenti di IA ad accedere alla nostra documentazione:

- [llms.txt](/llms.txt) - Contiene una panoramica strutturata dei file LLMs.txt di Aptos
- [llms-small.txt](/llms-small.txt) - Fornisce una versione condensata della documentazione, ottimizzata per finestre di contesto più piccole
- [llms-full.txt](/llms-full.txt) - Fornisce la documentazione completa e comprensiva per tutti i concetti di Aptos

## Utilizzo con Strumenti di IA

### Cursor

Usa la funzionalità `@Docs` in Cursor per includere i file LLMs.txt nel tuo progetto. Questo aiuta Cursor a fornire suggerimenti
di codice e documentazione più accurati per lo sviluppo di Aptos.

[Leggi di più su @Docs in Cursor](https://docs.cursor.com/context/@-symbols/@-docs)

### GitHub Copilot

GitHub Copilot può sfruttare le informazioni in questi file LLMs.txt per fornire un'assistenza migliore nello sviluppo di applicazioni Aptos. Puoi fare riferimento a questi file nella tua GitHub Copilot Chat utilizzando i seguenti URL:

```
https://aptos.dev/llms.txt
https://aptos.dev/llms-small.txt
https://aptos.dev/llms-full.txt
```

### Windsurf

Fai riferimento ai file LLMs.txt usando `@` o nei tuoi file `.windsurfrules` per migliorare la comprensione di Windsurf dello sviluppo di Aptos.

[Leggi di più su Windsurf Memories](https://docs.codeium.com/windsurf/memories#memories-and-rules)

### Altri Strumenti di IA

Qualsiasi strumento di IA che supporta LLMs.txt può utilizzare queste route per comprendere meglio Aptos. Semplicemente punta il tuo strumento a una qualsiasi delle
route sopra elencate.
