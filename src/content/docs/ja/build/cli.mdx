Aptosコマンドラインインターフェース（CLI）は、Moveコントラクトのコンパイルとテストを支援するツールです。また、Aptosの機能をオンチェーンで素早く試すのにも役立ちます。

上級ユーザーにとって、CLIはプライベートAptosネットワークの実行（ローカルでコードテストを支援）にも使用でき、ネットワークノードの管理にも役立ちます。
