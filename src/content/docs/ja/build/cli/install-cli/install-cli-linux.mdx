Linuxの場合,Aptos CLIツールをインストールする最も簡単な方法はPythonスクリプトを使用することですが,それが動作しない場合は,プリコンパイル済みバイナリをダウンロードして手動でインストールすることもできます.プリコンパイル済みバイナリによるアプローチは,アップデートが非常に手動的になるため,一般的にはお勧めしません.

<Aside type="caution">
  注意:ARMアーキテクチャを使用している場合は,こちらの手順に従ってインストールする必要があります:[特定バージョンのAptos CLIのインストール(上級者向け)](/ja/build/cli/install-cli/install-cli-specific-version)
</Aside>

# Pythonスクリプトを使用したインストール

<Steps>
  1. Python 3.6以上がインストールされていることをpython3 --versionを実行して確認します.

     python3がインストールされていない場合は,[python.org](http://python.org)でインストール手順を確認できます.

  2. ターミナルで,以下のコマンドのいずれかを使用します:

     ```shellscript filename="Terminal"
     curl -fsSL "https://aptos.dev/scripts/install_cli.py" | python3
     ```

     または,同等の`wget`コマンドを使用します:

     ```shellscript filename="Terminal"
     wget -qO- "https://aptos.dev/scripts/install_cli.py" | python3
     ```

     <Aside type="caution">
       `Couldn't find distutils or packaging. We cannot check the current version of the CLI. We will install the latest version.`というエラーが表示される場合は,`pip3 install packaging`を実行してから,この手順を繰り返してください.
     </Aside>

  3. (オプション)Aptos CLIをPATHの通ったフォルダに追加するか,直接PATHに追加すると便利です.

     - PATHにフォルダを追加する手順はシェルによって異なります.
     - `echo $SHELL`を実行してマシンのデフォルトシェルを確認し,そのシェルでフォルダをPATHに追加する具体的な手順を検索してください.

  4. 新しいターミナルを開いてaptos helpを実行し,スクリプトがインストールされていることを確認します

     - CLIで実行できるコマンドのリストが表示されるはずです.
     - 今後,各コマンドの詳細な使い方を知るのに役立ちます.
</Steps>

<Aside type="note" emoji="ℹ️">
  Aptos CLIを最新バージョンに更新したい場合は,`aptos update`を実行できます.
</Aside>

# プリコンパイル済みバイナリを使用したインストール(代替方法)

<Steps>
  1. に移動します.

  2. 最新リリースの"Assets"を展開してプリコンパイル済みバイナリを確認します.

  3. Linux用のzipファイルをダウンロードします.

     1. `aptos-cli-<バージョン>-Ubuntu-x86_64.zip`のような名前になっています.
     2. お使いのコンピュータのアーキテクチャに合った正しいzipファイルを選んでください.
     3. ダウンロード時に不審なファイルという警告が表示される可能性がありますが,無視して構いません.

  4. ダウンロードしたファイルを解凍します.

  5. 抽出したAptosバイナリファイルを任意のフォルダに移動します.

  6. ターミナルを開き,任意のフォルダに移動します.

  7. chmod +x ~/aptosを実行して~/aptosを実行可能にします.

  8. \~/aptos helpを実行してインストールしたバージョンが動作することを確認します.

     すべてのCLIコマンドの使用方法が表示されるはずです.これらは今後,特定のコマンドの使い方を理解する際に役立ちます.

  9. (オプション)Aptos CLIをPATHの通ったフォルダに追加するか,直接PATHに追加すると便利です.

     - PATHにフォルダを追加する手順はシェルによって異なります.
     - `echo $SHELL`を実行してマシンのデフォルトシェルを確認し,そのシェルでフォルダをPATHに追加する具体的な手順を検索してください.
</Steps>

<Aside type="note" emoji="ℹ️">
  プリコンパイル済みバイナリを使用してインストールした場合,Aptos CLIを更新するには,既存のインストールを削除してから,インストール手順を再度実行します.
</Aside>
