Macの場合,Aptos CLIをインストールする最も簡単な方法は,パッケージマネージャー`brew`を使用することです.

# インストール

<Steps>
  1. brewがインストールされていることを確認してください https://brew.sh/

  2. 新しいターミナルを開き,以下のコマンドを入力します.

     ```shellscript filename="Terminal"
     brew update
     brew install aptos
     ```

  3. 別のターミナルを開き,aptos helpを実行してCLIがインストールされていることを確認します.

     ```shellscript filename="Terminal"
     aptos help
     ```
</Steps>

<Aside type="caution">
  `brew`が動作しない場合は,こちらの手順を試してください:[特定バージョンのAptos CLIのインストール(上級者向け)](/ja/build/cli/install-cli/install-cli-specific-version)
</Aside>

# CLIのアップグレード

brewでCLIをアップグレードするには,2つのコマンドだけで済みます:

```shellscript filename="Terminal"
brew update
brew upgrade aptos
```
