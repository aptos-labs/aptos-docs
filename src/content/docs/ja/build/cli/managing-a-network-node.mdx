---
title: "Aptos CLIを使用したネットワークノードの管理"
---

[バリデータノードまたはバリデータフルノード(VFN)](/ja/network/nodes/validator-node)を実行している場合,CLIを使用してノードを操作することができます.

具体的には,以下のことができます:

1. [所有するステーキングプールの管理](/ja/network/nodes/validator-node/connect-nodes/staking-pool-operations)
2. [提案への投票](/ja/network/nodes/validator-node/connect-nodes/staking-pool-voter)

それ以外にも,ノードの操作に関連する特殊なコマンドについては,以下のヘルプコマンドを実行して確認できます:

```shellscript filename="Terminal"
aptos node --help
```
