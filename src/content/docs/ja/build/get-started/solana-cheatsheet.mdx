違いと類似点の詳細については[Aptos Learn](https://learn.aptoslabs.com/en/tutorials/solana-to-aptos-guide/cheat-sheet?workshop=solana-to-aptos)を参照してください

|                              | Solana                                                                                                             | Aptos                                                                 |
|------------------------------|--------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------|
| **スマートコントラクト**     | Rust, SVM                                                                                                          | Move, MoveVM                                                          |
| **トランザクション手数料**   | 低い                                                                                                               | 低い                                                                  |
| **並列化**                   | 悲観的並列化、すべての書き込みアカウントを宣言する必要                                                           | 楽観的並列化、チェーンがあなたのために書き込みアカウントを推論        |
| **コントラクトアカウントサポート** | PDAアカウント                                                                                                      | [オブジェクト](/build/smart-contracts/objects)または[リソースアカウント](/build/smart-contracts/resource-accounts)(代わりにオブジェクトの使用を推奨) |
| **データストレージ**         | プログラムが所有するアカウントにデータ保存                                                                         | ユーザーアカウントまたはオブジェクトにリソースとしてデータ保存         |
| **ストレージレベル**         | プログラムレベル                                                                                                   | オブジェクトに保存される場合グローバル                                |
| **ストレージマインドセット** | ユーザーデータがアカウントに分散して保存                                                                           | ユーザーデータがオブジェクトに分散して保存                            |
| **サンプルコード**           | [SolanaのTodoリストコントラクト](https://github.com/aptos-labs/move-by-examples/tree/main/advanced-todo-list/solana) | [AptosのTodoリストコントラクト](https://github.com/aptos-labs/move-by-examples/tree/main/advanced-todo-list/aptos) |
| **呼び出し元ID**            | `signer`                                                                                                           | `signer`                                                              |
| **アップグレード可能性**     | プログラムがアップグレード可能                                                                                     | モジュールがアップグレード可能                                        |
| **ディスパッチタイプ**       | 静的ディスパッチ                                                                                                   | 静的ディスパッチ                                                       |
| **FT標準**                   | トークンプログラム                                                                                                 | [Coin](/build/smart-contracts/aptos-coin) (legacy) と [Fungible Asset標準](/build/smart-contracts/fungible-asset) |
| **NFT標準**                  | トークンプログラム                                                                                                 | [Digital Asset標準](/build/smart-contracts/digital-asset)             |
| **ブロックチェーン相互作用** | Solana web3.jsライブラリ                                                                                           | [Aptos TypeScript SDK](/build/sdks/ts-sdk)                            |
