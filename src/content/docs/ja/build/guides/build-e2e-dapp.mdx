新しいフレームワークやプログラミング言語を学ぶ一般的な方法は,シンプルなTodoリストを作ることです.このチュートリアルでは,スマートコントラクトの側面からフロントエンド側まで,そして最後にウォレットを使用して両者を連携させる,エンドツーエンドのTodoリストDappの構築方法を学びます.

完成したコードは[my_first_dapp](https://github.com/aptos-labs/aptos-core/tree/main/aptos-move/move-examples/my_first_dapp)で確認できます.

## 前提条件

以下が必要です:

- [Aptos CLI](/ja/build/cli)
- [Aptos TypeScript SDK](/ja/build/sdks/ts-sdk)
- [Aptos Wallet Adapter](/ja/build/sdks/wallet-adapter)
- [Create React App](https://create-react-app.dev/)
- [node と npm](https://nodejs.org/en/)

Reactに関する決定事項についていくつか説明しますが,Reactの動作について深く掘り下げることはしません.そのため,Reactについてある程度の経験があることを前提としています.

## セットアップ

このセクションでは,クライアント側のコード(React ベース)とMoveコード(スマートコントラクト)の両方のプロジェクトファイルを格納する`my-first-dapp`ディレクトリを作成します.

1. ターミナルを開き,プロジェクトを作成したい目的のディレクトリに移動します(例:`Desktop`ディレクトリ).
2. `my-first-dapp`という新しいディレクトリを作成します:

```shellscript filename="Terminal"
mkdir my-first-dapp
```

3. そのディレクトリに移動します:

```shellscript filename="Terminal"
cd my-first-dapp
```

それでは[スマートコントラクトを作成](/ja/build/guides/build-e2e-dapp/1-create-smart-contract)しましょう.
