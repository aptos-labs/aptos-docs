.NETアプリケーションにAptosのWeb3機能を統合します.このSDKの目的は,互換性のあるゲームエンジンやプラットフォーム全体でマルチプラットフォームアプリケーションを構築するための開発者向けツールセットを提供することです.

**サポートされている機能**

- バイナリ正規シリアライゼーション(BCS)のエンコードとデコード
- Ed25519,SingleKey,MultiKey,およびKeylessサイナーのサポート
- トランザクションの構築,署名,送信のためのユーティリティ
- Aptos FullnodeとIndexer APIの抽象化
- Aptos Names(ANS)の解決と検索のサポート

## インストール

.NET SDKは[NuGet](https://www.nuget.org/packages/Aptos)で入手可能です.

以下のコマンドを使用して.NET SDKをインストールできます:

```shellscript
dotnet add package Aptos
```
