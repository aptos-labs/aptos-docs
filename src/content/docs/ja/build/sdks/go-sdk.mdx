## Go SDKのインストール

Aptosは[aptos-go-sdk GitHub](https://github.com/aptos-labs/aptos-go-sdk)リポジトリで公式のGo SDKを提供しています.Go SDKを使用するには,以下のように`Cargo.toml`でGitリポジトリから直接依存関係とパッチを追加してください:

import { CardGrid, LinkCard } from '@astrojs/starlight/components';

```shellscript
go get github.com/aptos-labs/aptos-go-sdk
```
