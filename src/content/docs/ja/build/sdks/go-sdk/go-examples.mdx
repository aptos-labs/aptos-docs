### サンプルの実行方法

サンプルスクリプトを実行するには:

<Steps>
  1. をクローンします

     ```shellscript filename="Terminal"
     git clone https://github.com/aptos-labs/aptos-go-sdk.git
     ```

  2. パッケージのトップレベルから,すべての依存関係をインストールします.

     ```shellscript filename="Terminal"
     go install
     ```

  3. パッケージをビルドします.

     ```shellscript filename="Terminal"
     go build ./...
     ```

  4. サンプルを実行します

     ```shellscript filename="Terminal"
     go run examples/transfer_coin/main.go
     ```
</Steps>
