### サンプルの実行方法

サンプルスクリプトを実行するには:

<Steps>
  1. をクローン

     ```shellscript filename="Terminal"
     git clone https://github.com/aptos-labs/aptos-ts-sdk.git
     ```

  2. パッケージのトップレベルから,すべての依存関係をインストール

     ```shellscript filename="Terminal"
     pnpm install
     ```

  3. パッケージをビルド

     ```shellscript filename="Terminal"
     pnpm build
     ```

  4. 実行したいサンプルのフォルダに移動

     ```shellscript filename="Terminal"
     cd examples/typescript
     ```

  5. ローカルの依存関係をインストール

     ```shellscript filename="Terminal"
     pnpm install
     ```

  6. サンプルを実行

     ```shellscript filename="Terminal"
     pnpm run simple_transfer
     ```
</Steps>
