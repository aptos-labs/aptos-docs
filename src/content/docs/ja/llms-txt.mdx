## LLMs.txtとは？

私たちは大規模言語モデル（LLMs）向けにAptosドキュメントを利用可能にする[LLMs.txt](https://llmstxt.org/)ファイルをサポートしています。
この機能により、AIツールがAptosブロックチェーン、Moveプログラミング言語、SDK、開発パターンをより良く理解できるようになります。

## 利用可能なルート

AIツールが私たちのドキュメントにアクセスできるよう、以下のLLMs.txtルートを提供しています：

- [llms.txt](/llms.txt) - Aptos LLMs.txtファイルの構造化された概要を含む
- [llms-small.txt](/llms-small.txt) - より小さなコンテキストウィンドウに最適化された、圧縮版のドキュメントを提供
- [llms-full.txt](/llms-full.txt) - すべてのAptosコンセプトに関する包括的な完全ドキュメントを提供

## AIツールでの使用法

### Cursor

CursorのLLMs.txtファイルをプロジェクトに含めるには、`@Docs`機能を使用してください。これにより、CursorがAptos開発でより正確なコード提案とドキュメントを提供できるようになります。

[CursorでのLLMの@Docsについて詳しく読む](https://docs.cursor.com/context/@-symbols/@-docs)

### GitHub Copilot

GitHub Copilotは、これらのLLMs.txtファイルの情報を活用してAptosアプリケーション開発時により良いサポートを提供できます。以下のURLを使用してGitHub Copilot Chatでこれらのファイルを参照できます：

```
https://aptos.dev/llms.txt
https://aptos.dev/llms-small.txt
https://aptos.dev/llms-full.txt
```

### Windsurf

WindsurfのAptos開発の理解を向上させるために、`@`を使用するか、`.windsurfrules`ファイルでLLMs.txtファイルを参照してください。

[Windsurf Memoriesについて詳しく読む](https://docs.codeium.com/windsurf/memories#memories-and-rules)

### その他のAIツール

LLMs.txtをサポートする任意のAIツールが、これらのルートを使用してAptosをより良く理解できます。単純に上記のいずれかのルートをツールに指定してください。
