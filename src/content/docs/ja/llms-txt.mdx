## LLMs.txtとは何ですか？

私たちは大きな言語モデル（LLMs）がAptosドキュメントを利用できるように[LLMs.txt](https://llmstxt.org/)ファイルをサポートしています。この機能は、AIツールがAptosブロックチェーン、Move言語、SDK、開発パターンをよりよく理解するのに役立ちます。

## 利用可能なルート

AIツールが私たちのドキュメントにアクセスするのを助けるために、以下のLLMs.txtルートを提供します：

- [llms.txt](/llms.txt) - Aptos LLMs.txtファイルの構造化された概要を含む
- [llms-small.txt](/llms-small.txt) - 小さいコンテキストウィンドウ向けに最適化されたドキュメントの凝縮版を提供
- [llms-full.txt](/llms-full.txt) - すべてのAptos概念に対する完全で包括的なドキュメントを提供

## AIツールとの使用

### Cursor

Cursorの`@Docs`機能をプロジェクトにLLMs.txtファイルを含めるために使用してください。これにより、CursorはAptos開発のためのより正確なコード提案とドキュメントを提供できます。

[Cursorの@Docsについて詳しく読む](https://docs.cursor.com/context/@-symbols/@-docs)

### GitHub Copilot

GitHub Copilotは、これらのLLMs.txtファイルの情報を活用して、Aptosアプリケーション開発時により良い支援を提供できます。以下のURLを使用してGitHub Copilot Chatでこれらのファイルを参照できます：

```
https://aptos.dev/llms.txt
https://aptos.dev/llms-small.txt
https://aptos.dev/llms-full.txt
```

### Windsurf

`@`を使用するか、`.windsurfrules`ファイルでLLMs.txtファイルを参照して、WindsurfのAptos開発理解を向上させてください。

[Windsurf Memoriesについて詳しく読む](https://docs.codeium.com/windsurf/memories#memories-and-rules)

### その他のAIツール

LLMs.txtをサポートするあらゆるAIツールは、これらのルートを使用してAptosをよりよく理解できます。ツールを上記のルートのいずれかに向けるだけで十分です。
