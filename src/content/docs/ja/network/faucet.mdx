Aptos Testnet Faucetにより、開発者は**Aptos testnet**で無料APTトークンを即時リクエストできます。これらのトークンは**テスト目的のみ**で使用され、実際の価値を持たず、mainnet APTと交換できません。

Faucetを使用して：

- **Moveスマートコントラクト**の展開とテスト
- Aptos SDKの実験 (TypeScript, Python, Rustなど)
- mainnetリリース前にdAppの構築と改善

<Faucet />

## Faucetの使用方法

1. **Googleアカウントでサインイン**します。
2. **Aptos testnetウォレットアドレスを入力**します。
3. **APTリクエスト**をクリックして即時トークンを受信します。
4. 数秒以内にウォレット残高を確認してください。

> **ヒント：** testnetアカウントがない場合は、[Aptos TypeScript SDKでアカウント作成](/build/sdks/ts-sdk/quickstart)(開発者向け)または[AptosウォレットインストールとTestnetへの切り替え](https://aptosfoundation.org/ecosystem/projects/wallets)(UI向け)から始めることができます。
