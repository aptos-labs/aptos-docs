以下のガイドでは,AptosネットワークにPFNをデプロイするためのステップバイステップの手順を説明します.

<Aside type="note">
  **PFNを実行する必要がありますか?**<br />
  PFNを実行したくないけれどもAptosブロックチェーンと対話したい場合は,Aptos LabsのPFNが提供するREST APIを使用できます([Aptos APIs](/ja/build/apis)を参照).ただし,Aptos Labsが提供するPFNにはレート制限があり,開発の妨げになる可能性があることに注意してください.このようなレート制限を回避したい場合は,独自のPFNをデプロイしてAptosブロックチェーンと直接同期することができます.
</Aside>

お好みのデプロイ方法のガイドを選択してください:

<Aside type="caution">
  **ネットワークを選択**<br />
  これらのガイドはデフォルトでAptosメインネットネットワークにPFNをデプロイしますが,他のネットワーク(例:devnetやtestnetネットワーク)でも同様に実行できます.そのためには,目的のネットワークブランチをチェックアウトし,それぞれのブランチの`genesis.blob`と`waypoint.txt`ノードファイルを使用してください:
  [メインネット](/ja/network/nodes/configure/node-files-all-networks/node-files-mainnet),
  [devnet](/ja/network/nodes/configure/node-files-all-networks/node-files-devnet),
  [testnet](/ja/network/nodes/configure/node-files-all-networks/node-files-testnet).
</Aside>
