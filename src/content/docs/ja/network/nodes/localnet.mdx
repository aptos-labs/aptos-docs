## 推奨

ほとんどの開発者は,CLIを使用してローカル開発ネットワークを実行することをお勧めします.
単一のローカルネット + フォーセットよりもシンプルで機能が充実しています.
本番環境のネットワークと同じように動作するローカルスタック
([Node API](/ja/build/apis) + [Transaction Stream](/ja/build/indexer/txn-stream) +
[Indexer API](/ja/build/indexer) + フォーセット)が必要な場合は,こちらを選択してください.

- ### [ローカル開発ネットワークを実行](/ja/network/nodes/localnet/local-development-network)

## 特殊な用途

Aptosノードソフトウェア自体を開発している場合や,複雑なテストが必要な場合は,このガイドを参照してください.

- ### [バリデーター付きローカルネットを実行](/ja/network/nodes/localnet/run-a-localnet)
