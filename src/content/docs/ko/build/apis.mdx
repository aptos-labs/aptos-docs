## Aptos 풀노드

이 API는 풀노드에 내장되어 있으며, Aptos 블록체인에서 상태를 _읽고_ 거래를 _제출하는_ 간단하고 낮은 지연시간의 저수준 방법을 제공합니다. 또한 거래 시뮬레이션도 지원합니다.

위에서 언급한 각 API의 코드는 [GitHub](https://github.com/aptos-labs/aptos-core)에서 오픈 소스로 공개되어 있습니다. 따라서 누구나 이러한 API를 운영할 수 있으며 전 세계의 많은 독립적인 운영자와 개발자들이 그렇게 하기로 선택합니다.

### Aptos Labs 운영 API 배포

[Aptos Labs](https://aptoslabs.com)는 [Aptos Foundation](https://aptosfoundation.org/)을 대신하여 각 [Aptos 네트워크](/ko/network/nodes/networks)에 대해 이러한 API의 배포를 운영하고 공개 사용을 위해 제공합니다.

이러한 API는 API 키 없이(익명 액세스) IP당 제한된 액세스를 허용합니다. 훨씬 높은 속도 제한을 받으려면 [Aptos Build](https://build.aptoslabs.com/) 계정에 가입할 수 있습니다.
