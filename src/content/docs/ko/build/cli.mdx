Aptos 명령줄 인터페이스(CLI)는 Move 컨트랙트를 컴파일하고 테스트하는 데 도움이 되는 도구입니다. 또한 온체인에서 Aptos 기능을 빠르게 경험해볼 수 있도록 도와줍니다.

고급 사용자의 경우, CLI는 개인 Aptos 네트워크를 실행하는 데도 사용할 수 있으며(로컬에서 코드 테스트에 도움), 네트워크 노드 관리에도 유용할 수 있습니다.
