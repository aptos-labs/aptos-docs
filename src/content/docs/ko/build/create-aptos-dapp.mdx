`create-aptos-dapp`은 dapp 개발자가 Aptos 네트워크에서 프론트엔드와 스마트 컨트랙트를 쉽게 만들 수 있도록 템플릿 프로젝트를 구축합니다.

## create-aptos-dapp을 사용하는 이유는?

- **템플릿 설정**: `create-aptos-dapp`은 미리 정의된 엔드투엔드 dapp 템플릿과 구성 파일을 생성합니다. 시간이 많이 걸리고 오류가 발생하기 쉬운 프로젝트 구조의 수동 설정을 절약해줍니다.
- **컨트랙트 디렉토리**: `create-aptos-dapp`은 Move 스마트 컨트랙트 모듈의 기본 구조를 포함하는 `contract` 디렉토리를 생성합니다.
- **모범 사례**: `create-aptos-dapp`은 Aptos 네트워크를 위한 개발 모범 사례와 구조 권장 사항을 통합합니다.
- **내장 Move 명령어**: `create-aptos-dapp`은 Move 컴파일러 초기화, 컴파일, 온체인 스마트 컨트랙트 게시와 같은 일반적인 작업을 위한 내장 명령어를 포함합니다.

## 전제 조건

- [node와 npm](https://nodejs.org/en) (npm ≥ 5.2.0)
- [Python 3.6+](https://www.python.org/)

## 템플릿

`create-aptos-dapp`은 미리 만들어진 엔드투엔드 dapp 템플릿, 즉 Aptos에서 dapp 생성을 시작할 수 있도록 구성과 아름다운 UI가 포함된 준비된 dapp을 제공합니다.

템플릿의 목표는 다음과 같습니다:

1. 엔드투엔드 dapp 템플릿 예제를 통해 다양한 Aptos 표준에 사용자를 익숙하게 하는 것
2. 프론트엔드 레이어에서 스마트 컨트랙트 레이어까지 Aptos에서 dapp을 구축하는 방법과 그 사이의 모든 것을 사용자에게 교육하는 것
3. 사용자에게 간단한 dapp을 빠르게 배포할 수 있는 미리 만들어진 템플릿 제공

## `create-aptos-dapp`이 활용하는 도구

- React 프레임워크
- Vite 개발 도구
- 스타일링을 위한 shadcn/ui + tailwind
- Aptos TS SDK
- Aptos Wallet Adapter
- Node 기반 Move 명령어
