개발 유형에 따라 환경을 설정하는 쉬운 방법입니다.

<Tabs>
  {/* Frontend */}

  <TabItem label="프론트엔드">
    <Steps>
      1. 프론트엔드 프로젝트 초기화

         인기 있는 선택의 몇 가지 예시입니다:

         <Tabs>
           <TabItem label="Next.js">
             ```shellscript filename="Terminal"
             pnpx create-next-app@latest
             ```
           </TabItem>

           <TabItem label="Vite (TS)">
             ```shellscript filename="Terminal"
             pnpx create vite my-aptos-app --template react-ts
             ```
           </TabItem>
         </Tabs>

      2. @aptos-labs/ts-sdk 설치

         ```shellscript npm2yarn
         npm i @aptos-labs/ts-sdk
         ```

      3. TS SDK 설정

         <CardGrid>
           <LinkCard
             href="/build/sdks/ts-sdk/quickstart"
             title="TS SDK 빠른 시작"
             description="계정, 네트워크 설정 방법, faucet 사용,
         트랜잭션 전송/시뮬레이션 등을 확인하세요"
           />
         </CardGrid>

      4. 앱을 구축하세요!

         프론트엔드에서 Aptos를 사용하기 위한 개발자 설정이 이제 완료되었습니다.
         개발 프로세스를 간소화하는 다른 도구들을 확인하세요

         <CardGrid>
           <LinkCard
             href="/build/indexer"
             title="인덱서"
             description="잔액, 트랜잭션 활동, 토큰 데이터 등과 같은
         온체인 상태를 효율적으로 쿼리하세요"
           />

           <LinkCard
             href="https://github.com/aptos-labs/aptos-ts-sdk/tree/main/examples/typescript"
             title="TS SDK 예제"
             description="TS SDK 사용 방법에 대한
         20+ 예제"
             target="_blank"
           />

           <LinkCard href="https://geomi.dev/" title="Geomi" description="Fullnode API / 인덱서에 대한 속도 제한에 도달하셨나요? 여기서 API 키를 받으세요" target="_blank" />
         </CardGrid>
    </Steps>
  </TabItem>

  {/* Smart Contract */}

  <TabItem label="스마트 컨트랙트">
    <Steps>
      1. CLI 설치

         <CardGrid>
           <LinkCard href="/build/cli" title="Aptos CLI" description="Aptos CLI 설치 지침" />
         </CardGrid>

      2. 편집기 또는 IDE 설정

         선택한 편집기에 다음 확장을 추가하여 Move 개발을 더 쉽게 만드세요

         <Tabs>
           <TabItem label="JetBrains IDE">
             <CardGrid>
               <LinkCard
                 href="https://plugins.jetbrains.com/plugin/14721-move-language"
                 title="Move on Aptos"
                 description="CLion, Rust Rover, WebStorm과 같은
         JetBrains IDE를 위한 언어 서버 및 구문 강조기,
         IntelliJ"
                 target="_blank"
               />
             </CardGrid>
           </TabItem>

           <TabItem label="VSCode">
             <CardGrid>
               <LinkCard href="https://marketplace.visualstudio.com/items?itemName=AptosLabs.move-on-aptos" title="Move on Aptos 확장" description="VSCode를 위한 완전한 Aptos Move IDE" target="_blank" />
             </CardGrid>
           </TabItem>

           <TabItem label="Cursor, Kiro 등">
             <CardGrid>
               <LinkCard href="https://open-vsx.org/extension/aptoslabs/move-on-aptos" title="Move on Aptos 확장" description="OpenVSX 기반 플랫폼을 위한 완전한 Aptos Move IDE" target="_blank" />
             </CardGrid>
           </TabItem>
         </Tabs>

      3. 스마트 컨트랙트 생성

         애플리케이션 폴더로 이동하여 다음을 수행하여 새 스마트 컨트랙트를 초기화하세요:

         ```shellscript filename="Terminal"
         aptos move init --name my_todo_list
         ```

      4. 스마트 컨트랙트 구축, 컴파일 및 배포!

         스마트 컨트랙트에 Aptos를 사용하기 위한 개발자 설정이 이제 완료되었습니다.
         자세한 정보는 아래 Dapp 튜토리얼 링크를 참조하세요

         <CardGrid>
           <LinkCard
             href="/build/guides/build-e2e-dapp/1-create-smart-contract#what-is-a-movetoml-file"
             title="스마트 컨트랙트 생성 가이드"
             description="Move로 스마트 컨트랙트를 설정하기 위한
         쉬운 할 일 목록 가이드"
           />
         </CardGrid>
    </Steps>
  </TabItem>

  {/* create-aptos-dapp */}

  <TabItem label="Aptos DApp 생성">
    <Steps>
      1. create-aptos-dapp 설치

         몇 초 안에 템플릿에서 dApp을 설치하려면 아래 명령을 실행하세요:

         <Tabs>
           { /* npx */ }

           <TabItem label="npx">
             ```shellscript filename="Terminal"
             npx create-aptos-dapp@latest
             ```
           </TabItem>

           { /* pnpx */ }

           <TabItem label="pnpx">
             ```shellscript filename="Terminal"
             pnpx create-aptos-dapp@latest
             ```
           </TabItem>
         </Tabs>

      2. 프롬프트를 따르세요

         새 dApp의 이름, [템플릿](/build/create-aptos-dapp#templates) 및 네트워크를 선택하려면 CLI 프롬프트를 따르세요.

         ![cad](~/images/cad-video.gif)

      3. 새 dApp을 구축하고 사용자 정의하기 시작하세요!

         새 프로젝트로 이동하여 구축을 계속하려면 선호하는 IDE에서 열어보세요!

         다음 단계는 생성된 `README.md` 파일을 따르세요.

      4. 계속 읽어보세요

         </CardGrid>
    </Steps>
  </TabItem>
