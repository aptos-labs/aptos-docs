---
title: "Solana에서 Aptos로 치트시트"
---

차이점과 유사점에 대해 자세히 알아보려면 [Aptos Learn](https://learn.aptoslabs.com/en/tutorials/solana-to-aptos-guide/cheat-sheet?workshop=solana-to-aptos)을 참조하세요

|                              | Solana                                                                                                             | Aptos                                                                                                                                     |
| ---------------------------- | ------------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------- |
| **스마트 컨트랙트**          | Rust, SVM                                                                                                          | Move, MoveVM                                                                                                                              |
| **거래 수수료**         | 낮음                                                                                                                | 낮음                                                                                                                                       |
| **병렬화**          | 비관적 병렬성, 모든 쓰기 계정을 선언해야 함                                                        | 낙관적 병렬성, 체인이 쓰기 계정을 추론해줌                                                                               |
| **컨트랙트 계정 지원** | PDA 계정                                                                                                        | [객체](/ko/build/smart-contracts/objects) 또는 [리소스 계정](/ko/build/smart-contracts/resource-accounts)(대신 객체 사용 권장) |
| **데이터 저장**             | 프로그램이 소유하는 계정에 데이터 저장                                                                           | 사용자 계정 또는 객체 하에 리소스로 데이터 저장                                                                                      |
| **저장 수준**            | 프로그램 수준                                                                                                      | 객체 하에 저장될 때 글로벌                                                                                                           |
| **저장 방식**          | 계정 하에 분산적으로 저장되는 사용자 데이터                                                                       | 객체 하에 분산적으로 저장되는 사용자 데이터                                                                                               |
| **예제 코드**             | [Solana의 Todo 리스트 컨트랙트](https://github.com/aptos-labs/move-by-examples/tree/main/advanced-todo-list/solana) | [Aptos의 Todo 리스트 컨트랙트](https://github.com/aptos-labs/move-by-examples/tree/main/advanced-todo-list/aptos)                          |
| **호출자 ID**                | `signer`                                                                                                           | `signer`                                                                                                                                  |
| **업그레이드 가능성**            | 프로그램은 업그레이드 가능                                                                                              | 모듈은 업그레이드 가능                                                                                                                      |
| **디스패치 타입**            | 정적 디스패치                                                                                                    | 정적 디스패치                                                                                                                           |
| **FT 표준**             | 토큰 프로그램                                                                                                      | [Coin](/ko/build/smart-contracts/aptos-coin) (레거시) 및 [대체 가능한 자산 표준](/ko/build/smart-contracts/fungible-asset)                   |
| **NFT 표준**            | 토큰 프로그램                                                                                                      | [디지털 자산 표준](/ko/build/smart-contracts/digital-asset)                                                                            |
| **블록체인 상호작용**   | Solana web3.js 라이브러리                                                                                             | [Aptos TypeScript SDK](/ko/build/sdks/ts-sdk)                                                                                                |
