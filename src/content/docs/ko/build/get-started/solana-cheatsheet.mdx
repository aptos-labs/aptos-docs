차이점과 유사점에 대해 자세히 알아보려면 [Aptos Learn](https://learn.aptoslabs.com/en/tutorials/solana-to-aptos-guide/cheat-sheet?workshop=solana-to-aptos)을 참조하세요

|                              | Solana                                                                                                             | Aptos                                                                 |
|------------------------------|--------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------|
| **스마트 컨트랙트**         | Rust, SVM                                                                                                          | Move, MoveVM                                                          |
| **트랜잭션 수수료**         | 낮음                                                                                                               | 낮음                                                                  |
| **병렬화**                   | 비관적 병렬화, 모든 쓰기 계정을 선언해야 함                                                                        | 낙관적 병렬화, 체인이 당신을 위해 쓰기 계정을 추론                    |
| **컨트랙트 계정 지원**      | PDA 계정                                                                                                           | [객체](/build/smart-contracts/objects) 또는 [리소스 계정](/build/smart-contracts/resource-accounts)(대신 객체 사용 권장) |
| **데이터 저장**             | 프로그램이 소유한 계정에 데이터 저장                                                                               | 사용자 계정이나 객체에 리소스로 데이터 저장                           |
| **저장 수준**               | 프로그램 수준                                                                                                      | 객체에 저장될 때 글로벌                                              |
| **저장 정신**               | 사용자 데이터가 계정에 분산되어 저장                                                                               | 사용자 데이터가 객체에 분산되어 저장                                  |
| **예제 코드**               | [Solana의 할 일 목록 컨트랙트](https://github.com/aptos-labs/move-by-examples/tree/main/advanced-todo-list/solana) | [Aptos의 할 일 목록 컨트랙트](https://github.com/aptos-labs/move-by-examples/tree/main/advanced-todo-list/aptos) |
| **호출자 ID**               | `signer`                                                                                                           | `signer`                                                              |
| **업그레이드 가능성**       | 프로그램이 업그레이드 가능                                                                                         | 모듈이 업그레이드 가능                                                |
| **디스패치 유형**           | 정적 디스패치                                                                                                      | 정적 디스패치                                                         |
| **FT 표준**                 | 토큰 프로그램                                                                                                      | [Coin](/build/smart-contracts/aptos-coin) (legacy) 및 [Fungible Asset 표준](/build/smart-contracts/fungible-asset) |
| **NFT 표준**                | 토큰 프로그램                                                                                                      | [Digital Asset 표준](/build/smart-contracts/digital-asset)           |
| **블록체인 상호작용**       | Solana web3.js 라이브러리                                                                                          | [Aptos TypeScript SDK](/build/sdks/ts-sdk)                            |
