## LLMs.txt란 무엇인가요?

우리는 Aptos 문서를 대규모 언어 모델(LLM)에 제공하기 위해 [LLMs.txt](https://llmstxt.org/) 파일을 지원합니다.
이 기능은 AI 도구가 Aptos 블록체인, Move 언어, SDK 및 개발 패턴을 더 잘 이해할 수 있도록 도와줍니다.

## 사용 가능한 경로

AI 도구가 우리 문서에 액세스할 수 있도록 다음 LLMs.txt 경로를 제공합니다:

- [llms.txt](/llms.txt) - Aptos LLMs.txt 파일의 구조화된 개요 포함
- [llms-small.txt](/llms-small.txt) - 더 작은 컨텍스트 창에 최적화된 압축된 문서 버전 제공
- [llms-full.txt](/llms-full.txt) - 모든 Aptos 개념에 대한 완전하고 포괄적인 문서 제공

## AI 도구와 함께 사용하기

### Cursor

Cursor의 `@Docs` 기능을 사용하여 프로젝트에 LLMs.txt 파일을 포함하세요. 이는 Cursor가 Aptos 개발을 위한 더 정확한
코드 제안과 문서를 제공하는 데 도움이 됩니다.

[Cursor의 @Docs에 대해 더 읽기](https://docs.cursor.com/context/@-symbols/@-docs)

### GitHub Copilot

GitHub Copilot은 이러한 LLMs.txt 파일의 정보를 활용하여 Aptos 애플리케이션 개발 시 더 나은 지원을 제공할 수 있습니다. 다음 URL을 사용하여 GitHub Copilot Chat에서 이러한 파일을 참조할 수 있습니다:

```
https://aptos.dev/llms.txt
https://aptos.dev/llms-small.txt
https://aptos.dev/llms-full.txt
```

### Windsurf

`@`를 사용하거나 `.windsurfrules` 파일에서 LLMs.txt 파일을 참조하여 Windsurf의 Aptos 개발 이해를 향상시키세요.

[Windsurf Memories에 대해 더 읽기](https://docs.codeium.com/windsurf/memories#memories-and-rules)

### 기타 AI 도구

LLMs.txt를 지원하는 모든 AI 도구는 이러한 경로를 사용하여 Aptos를 더 잘 이해할 수 있습니다. 단순히 도구를 위의
경로 중 하나로 가리키기만 하면 됩니다.
