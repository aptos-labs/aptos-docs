Aptos Testnet Faucet을 통해 개발자가 **Aptos testnet**에서 무료 APT 토큰을 즉시 요청할 수 있습니다. 이러한 토큰은 **테스트 목적으로만** 사용되며 실제 가치를 가지지 않으며 mainnet APT로 교환할 수 없습니다.

Faucet을 사용하여:

- **Move 스마트 컨트랙트** 배포 및 테스트
- Aptos SDK 실험 (TypeScript, Python, Rust 등)
- mainnet 출시 전에 dApp 구축 및 개선

<Faucet />

## Faucet 사용 방법

1. **Google 계정으로 로그인**하세요.
2. **Aptos testnet 지갑 주소를 입력**하세요.
3. **APT 요청**을 클릭하여 즉시 토큰을 받으세요.
4. 몇 초 안에 지갑 잔액을 확인하세요.

> **팁:** testnet 계정이 없는 경우 [Aptos TypeScript SDK로 계정 생성](/build/sdks/ts-sdk/quickstart)(개발자 방법) 또는 [Aptos 지갑 설치 및 Testnet으로 전환](https://aptosfoundation.org/ecosystem/projects/wallets)(UI 방법)을 통해 시작할 수 있습니다.
