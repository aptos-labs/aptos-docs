Para saber mais sobre as diferenças e semelhanças, consulte [Aptos Learn](https://learn.aptoslabs.com/en/tutorials/solana-to-aptos-guide/cheat-sheet?workshop=solana-to-aptos)

|                              | Solana                                                                                                             | Aptos                                                                 |
|------------------------------|--------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------|
| **Contratos Inteligentes**   | Rust, SVM                                                                                                          | Move, MoveVM                                                          |
| **Taxas de Transação**       | Baixas                                                                                                             | Baixas                                                                |
| **Paralelização**            | Paralelização pessimista, necessidade de declarar todas as contas de escrita                                      | Paralelização otimista, a cadeia infere contas de escrita para você   |
| **Suporte a Conta de Contrato** | Conta PDA                                                                                                          | [Objeto](/build/smart-contracts/objects) ou [conta de recurso](/build/smart-contracts/resource-accounts) (encoraje o uso de objeto em vez disso) |
| **Armazenamento de Dados**   | Dados armazenados em conta de propriedade de programas                                                             | Dados armazenados como recurso na conta do usuário ou objeto          |
| **Nível de Armazenamento**   | Nível de programa                                                                                                  | Global quando armazenado em objeto                                    |
| **Mentalidade de Armazenamento** | Dados do usuário armazenados de forma distribuída na conta                                                         | Dados do usuário armazenados de forma distribuída no objeto           |
| **Exemplo de Código**        | [Contrato de lista de tarefas no Solana](https://github.com/aptos-labs/move-by-examples/tree/main/advanced-todo-list/solana) | [Contrato de lista de tarefas no Aptos](https://github.com/aptos-labs/move-by-examples/tree/main/advanced-todo-list/aptos) |
| **ID do Chamador**           | `signer`                                                                                                           | `signer`                                                              |
| **Atualizabilidade**         | O programa é atualizável                                                                                           | O módulo é atualizável                                                |
| **Tipo de Dispatch**         | Dispatch estático                                                                                                  | Dispatch estático                                                     |
| **Padrões FT**               | Programa de token                                                                                                  | [Coin](/build/smart-contracts/aptos-coin) (legacy) e [Padrão de Ativo Fungível](/build/smart-contracts/fungible-asset) |
| **Padrões NFT**              | Programa de token                                                                                                  | [Padrão de Ativo Digital](/build/smart-contracts/digital-asset)       |
| **Interação Blockchain**     | Biblioteca Solana web3.js                                                                                          | [SDK TypeScript Aptos](/build/sdks/ts-sdk)                            |
