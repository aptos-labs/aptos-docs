## O que é LLMs.txt?

Nós oferecemos suporte a arquivos [LLMs.txt](https://llmstxt.org/) para tornar a documentação Aptos disponível para grandes modelos de linguagem (LLMs). Esse recurso ajuda ferramentas de IA a entender melhor a blockchain Aptos, sua linguagem Move, SDKs e padrões de desenvolvimento.

## Rotas Disponíveis

Fornecemos as seguintes rotas LLMs.txt para ajudar ferramentas de IA a acessar nossa documentação:

- [llms.txt](/llms.txt) - Contém uma visão geral estruturada dos arquivos LLMs.txt do Aptos
- [llms-small.txt](/llms-small.txt) - Fornece uma versão condensada da documentação, otimizada para janelas de contexto menores
- [llms-full.txt](/llms-full.txt) - Fornece a documentação completa e abrangente para todos os conceitos Aptos

## Uso com Ferramentas de IA

### Cursor

Use o recurso `@Docs` no Cursor para incluir os arquivos LLMs.txt em seu projeto. Isso ajuda o Cursor a fornecer sugestões de código mais precisas e documentação para desenvolvimento Aptos.

[Leia mais sobre @Docs no Cursor](https://docs.cursor.com/context/@-symbols/@-docs)

### GitHub Copilot

O GitHub Copilot pode aproveitar as informações nesses arquivos LLMs.txt para fornecer melhor assistência no desenvolvimento de aplicações Aptos. Você pode referenciar esses arquivos no seu GitHub Copilot Chat usando as seguintes URLs:

```
https://aptos.dev/llms.txt
https://aptos.dev/llms-small.txt
https://aptos.dev/llms-full.txt
```

### Windsurf

Referencie os arquivos LLMs.txt usando `@` ou em seus arquivos `.windsurfrules` para melhorar a compreensão do Windsurf sobre desenvolvimento Aptos.

[Leia mais sobre Windsurf Memories](https://docs.codeium.com/windsurf/memories#memories-and-rules)

### Outras Ferramentas de IA

Qualquer ferramenta de IA que suporte LLMs.txt pode usar essas rotas para entender melhor Aptos. Basta apontar sua ferramenta para qualquer uma das rotas acima.
