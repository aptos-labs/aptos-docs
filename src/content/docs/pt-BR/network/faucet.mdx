O Faucet Aptos Testnet permite que desenvolvedores solicitem instantaneamente tokens APT gratuitos na **rede testnet Aptos**. Estes tokens são apenas para **fins de teste** — eles não têm valor real e não podem ser trocados por APT mainnet.

Use o faucet para:

- Implantar e testar **contratos inteligentes Move**
- Experimentar com SDKs Aptos (TypeScript, Python, Rust, e mais)
- Construir e refinar dApps antes de lançar para mainnet

<Faucet />

## Como Usar o Faucet

1. **Entre** com sua conta Google.
2. **Digite seu endereço de carteira testnet Aptos**.
3. Clique em **Solicitar APT** para receber tokens instantaneamente.
4. Verifique o saldo da sua carteira — tokens devem aparecer em poucos segundos.

> **Dica:** Se você não tem uma conta testnet, você pode tanto [Criar uma Conta com o SDK TypeScript Aptos](/build/sdks/ts-sdk/quickstart) (método desenvolvedor) ou [Instalar uma Carteira Aptos e Mudar para Testnet](https://aptosfoundation.org/ecosystem/projects/wallets) (método UI) para começar.
