A interface de linha de comando (CLI) do Aptos é uma ferramenta para ajudar você a compilar e testar contratos Move. Também pode ajudar você a experimentar rapidamente os recursos do Aptos on-chain.

Para usuários mais avançados, o CLI também pode ser usado para executar uma rede Aptos privada (para ajudar a testar código localmente) e pode ser útil para gerenciar um nó de rede.
