O Faucet Aptos é uma ferramenta que permite que desenvolvedores obtenham tokens APT de teste para experimentar com o blockchain Aptos em redes de teste como devnet e testnet.

## Como usar o Faucet

<Faucet />

## Redes disponíveis

O faucet está disponível nas seguintes redes de teste:

- **Devnet**: Rede de desenvolvimento que é resetada regularmente
- **Testnet**: Rede de teste mais estável para desenvolvimento de aplicações

## Limites e restrições

- Cada endereço pode solicitar tokens apenas uma vez por período de tempo
- A quantidade de tokens por solicitação é limitada
- O faucet é apenas para fins de desenvolvimento e teste

## Usando o faucet programaticamente

Você também pode usar o faucet programaticamente através do SDK:

### TypeScript

```typescript
import { Aptos, AptosConfig, Network } from "@aptos-labs/ts-sdk";

const config = new AptosConfig({ network: Network.DEVNET });
const aptos = new Aptos(config);

// Financiar uma conta
await aptos.fundAccount({
  accountAddress: "0x123...",
  amount: 100_000_000, // 1 APT em octas
});
```

### Python

```python
from aptos_sdk.client import RestClient

client = RestClient("https://fullnode.devnet.aptoslabs.com/v1")
await client.fund_account(account_address, 100_000_000)
```

## Solução de problemas

Se você encontrar problemas ao usar o faucet:

1. Verifique se está usando o endereço correto
2. Certifique-se de que não excedeu os limites de taxa
3. Tente novamente após alguns minutos se a solicitação falhar

Para mais informações sobre como obter tokens de teste, consulte nossa documentação sobre [configuração de desenvolvimento](/pt/build/get-started/developer-setup).
