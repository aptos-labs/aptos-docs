Чтобы узнать больше о различиях и сходствах, см. [Aptos Learn](https://learn.aptoslabs.com/en/tutorials/solana-to-aptos-guide/cheat-sheet?workshop=solana-to-aptos)

|                              | Solana                                                                                                             | Aptos                                                                 |
|------------------------------|--------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------|
| **Смарт-контракты**          | Rust, SVM                                                                                                          | Move, MoveVM                                                          |
| **Комиссии за транзакции**   | Низкие                                                                                                             | Низкие                                                                |
| **Параллелизация**           | Пессимистическая параллелизация, нужно объявлять все аккаунты записи                                             | Оптимистическая параллелизация, цепочка выводит аккаунты записи за вас |
| **Поддержка аккаунта контракта** | PDA аккаунт                                                                                                       | [Объект](/build/smart-contracts/objects) или [аккаунт ресурсов](/build/smart-contracts/resource-accounts)(рекомендуется использовать объект вместо этого) |
| **Хранение данных**          | Данные хранятся в аккаунте, принадлежащем программам                                                              | Данные хранятся как ресурс в аккаунте пользователя или объекте         |
| **Уровень хранения**         | Уровень программы                                                                                                  | Глобальный при хранении в объекте                                     |
| **Мышление о хранении**      | Данные пользователей хранятся распределенно в аккаунте                                                             | Данные пользователей хранятся распределенно в объекте                  |
| **Пример кода**              | [Контракт списка задач в Solana](https://github.com/aptos-labs/move-by-examples/tree/main/advanced-todo-list/solana) | [Контракт списка задач в Aptos](https://github.com/aptos-labs/move-by-examples/tree/main/advanced-todo-list/aptos) |
| **ID вызывающего**           | `signer`                                                                                                           | `signer`                                                              |
| **Возможность обновления**   | Программа обновляема                                                                                              | Модуль обновляем                                                      |
| **Тип диспетчеризации**      | Статическая диспетчеризация                                                                                       | Статическая диспетчеризация                                           |
| **Стандарт FT**              | Программа токенов                                                                                                  | [Coin](/build/smart-contracts/aptos-coin) (legacy) и [Стандарт Fungible Asset](/build/smart-contracts/fungible-asset) |
| **Стандарты NFT**            | Программа токенов                                                                                                  | [Стандарт Digital Asset](/build/smart-contracts/digital-asset)        |
| **Взаимодействие с блокчейном** | Библиотека Solana web3.js                                                                                          | [Aptos TypeScript SDK](/build/sdks/ts-sdk)                            |
