## Что такое LLMs.txt?

Мы поддерживаем файлы [LLMs.txt](https://llmstxt.org/) для предоставления документации Aptos большим языковым моделям
(LLM). Эта функция помогает инструментам ИИ лучше понимать блокчейн Aptos, его язык Move, SDK и паттерны разработки.

## Доступные маршруты

Мы предоставляем следующие маршруты LLMs.txt для помощи инструментам ИИ в доступе к нашей документации:

- [llms.txt](/llms.txt) - Содержит структурированный обзор файлов LLMs.txt для Aptos
- [llms-small.txt](/llms-small.txt) - Предоставляет сжатую версию документации, оптимизированную для меньших контекстных окон
- [llms-full.txt](/llms-full.txt) - Предоставляет полную комплексную документацию для всех концепций Aptos

## Использование с инструментами ИИ

### Cursor

Используйте функцию `@Docs` в Cursor для включения файлов LLMs.txt в ваш проект. Это помогает Cursor предоставлять более точные
предложения кода и документацию для разработки Aptos.

[Подробнее о @Docs в Cursor](https://docs.cursor.com/context/@-symbols/@-docs)

### GitHub Copilot

GitHub Copilot может использовать информацию из этих файлов LLMs.txt для предоставления лучшей помощи при разработке приложений Aptos. Вы можете ссылаться на эти файлы в вашем GitHub Copilot Chat, используя следующие URL:

```
https://aptos.dev/llms.txt
https://aptos.dev/llms-small.txt
https://aptos.dev/llms-full.txt
```

### Windsurf

Ссылайтесь на файлы LLMs.txt используя `@` или в ваших файлах `.windsurfrules` для улучшения понимания Windsurf разработки Aptos.

[Подробнее о Windsurf Memories](https://docs.codeium.com/windsurf/memories#memories-and-rules)

### Другие инструменты ИИ

Любой инструмент ИИ, который поддерживает LLMs.txt, может использовать эти маршруты для лучшего понимания Aptos. Просто направьте ваш инструмент на любой из
маршрутов выше.
