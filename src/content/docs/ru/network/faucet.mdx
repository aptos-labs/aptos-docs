Кран Aptos Testnet позволяет разработчикам мгновенно запрашивать бесплатные токены APT на **сети testnet Aptos**. Эти токены предназначены только для **целей тестирования** — они не имеют реальной ценности и не могут быть обменяны на APT mainnet.

Используйте кран для:

- Развертывания и тестирования **смарт-контрактов Move**
- Экспериментирования с SDK Aptos (TypeScript, Python, Rust и больше)
- Построения и улучшения dApps перед запуском в mainnet

## Как использовать кран

1. **Войдите** со своим аккаунтом Google.
2. **Введите адрес вашего кошелька testnet Aptos**.
3. Нажмите **Запросить APT** для мгновенного получения токенов.
4. Проверьте баланс вашего кошелька — токены должны появиться в течение нескольких секунд.

> **Совет:** Если у вас нет аккаунта testnet, вы можете либо [Создать аккаунт с помощью Aptos TypeScript SDK](/build/sdks/ts-sdk/quickstart) (метод разработчика), либо [Установить кошелек Aptos и переключиться на Testnet](https://aptosfoundation.org/ecosystem/projects/wallets) (метод UI) для начала.
