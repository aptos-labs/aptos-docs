Geliştirme türüne bağlı olarak ortamınızı kurmanın kolay bir yolu burada.

<Tabs>
  {/* Frontend */}

  <TabItem label="Ön Uç">
    <Steps>
      1. Ön Uç Projesini Başlatma

         Popüler seçeneklerden birkaç örnek:

         <Tabs>
           <TabItem label="Next.js">
             ```shellscript filename="Terminal"
             pnpx create-next-app@latest
             ```
           </TabItem>

           <TabItem label="Vite (TS)">
             ```shellscript filename="Terminal"
             pnpx create vite my-aptos-app --template react-ts
             ```
           </TabItem>
         </Tabs>

      2. @aptos-labs/ts-sdk'yi yükleyin

         ```shellscript npm2yarn
         npm i @aptos-labs/ts-sdk
         ```

      3. TS SDK'yi ayarlayın

         <CardGrid>
           <LinkCard
             href="/build/sdks/ts-sdk/quickstart"
             title="TS SDK Hızlı Başlangıç"
             description="Hesabınızı, ağı nasıl ayarlayacağınızı öğrenin, faucet kullanın,
         işlemleri gönderin / simüle edin ve daha fazlası"
           />
         </CardGrid>

      4. Uygulamanızı oluşturun!

         Ön uçta Aptos'u kullanmak için geliştirici kurulumu artık tamamlandı.
         Geliştirme sürecini basitleştiren diğer araçlarımıza göz atın

         <CardGrid>
           <LinkCard
             href="/build/indexer"
             title="İndeksleyici"
             description="Bakiyeler, işlem aktivitesi, token verileri gibi
         zincir üzeri durumu verimli bir şekilde sorgulayın ve daha fazlası"
           />

           <LinkCard
             href="https://github.com/aptos-labs/aptos-ts-sdk/tree/main/examples/typescript"
             title="TS SDK Örnekleri"
             description="TS SDK'nin nasıl kullanılacağına dair
         20+ örnek"
             target="_blank"
           />

           <LinkCard href="https://geomi.dev/" title="Geomi" description="Fullnode API / İndeksleyiciler için hız sınırlarına ulaştınız mı? Burada bir API anahtarı alın" target="_blank" />
         </CardGrid>
    </Steps>
  </TabItem>

  {/* Smart Contract */}

  <TabItem label="Akıllı Sözleşme">
    <Steps>
      1. CLI'yi yükleyin

         <CardGrid>
           <LinkCard href="/build/cli" title="Aptos CLI" description="Aptos CLI'yi yükleme talimatları" />
         </CardGrid>

      2. Düzenleyiciyi veya IDE'yi ayarlayın

         Move geliştirmesini kolaylaştırmak için seçtiğiniz düzenleyiciye aşağıdaki uzantıları ekleyin

         <Tabs>
           <TabItem label="JetBrains IDE'leri">
             <CardGrid>
               <LinkCard
                 href="https://plugins.jetbrains.com/plugin/14721-move-language"
                 title="Move on Aptos"
                 description="CLion, Rust Rover, WebStorm gibi için
         dil sunucusu ve sözdizimi vurgulayıcı,
         IntelliJ"
                 target="_blank"
               />
             </CardGrid>
           </TabItem>

           <TabItem label="VSCode">
             <CardGrid>
               <LinkCard href="https://marketplace.visualstudio.com/items?itemName=AptosLabs.move-on-aptos" title="Move on Aptos uzantısı" description="VSCode için tam özellikli Aptos Move IDE" target="_blank" />
             </CardGrid>
           </TabItem>

           <TabItem label="Cursor, Kiro vb.">
             <CardGrid>
               <LinkCard href="https://open-vsx.org/extension/aptoslabs/move-on-aptos" title="Move on Aptos uzantısı" description="OpenVSX tabanlı platformlar için tam özellikli Aptos Move IDE" target="_blank" />
             </CardGrid>
           </TabItem>
         </Tabs>

      3. Akıllı Sözleşme Oluşturun

         Uygulama klasörünüze gidin ve aşağıdakileri yaparak yeni bir akıllı sözleşmeyi başlatın:

         ```shellscript filename="Terminal"
         aptos move init --name my_todo_list
         ```

      4. Akıllı Sözleşmeyi Oluşturun, Derleyin ve Dağıtın!

         Akıllı sözleşmeler için Aptos'u kullanmak üzere geliştirici kurulumu artık tamamlandı.
         Daha fazla bilgi için aşağıdaki Dapp öğretici bağlantısına bakın

         <CardGrid>
           <LinkCard
             href="/build/guides/build-e2e-dapp/1-create-smart-contract#what-is-a-movetoml-file"
             title="Akıllı Sözleşme Oluşturma Kılavuzu"
             description="Move ile akıllı sözleşme ayarlamak için
         basit bir yapılacaklar listesi kılavuzu"
           />
         </CardGrid>
    </Steps>
  </TabItem>

  {/* create-aptos-dapp */}

  <TabItem label="Aptos DApp Oluşturun">
    <Steps>
      1. create-aptos-dapp'i yükleyin

         Saniyeler içinde bir şablondan dApp yüklemek için aşağıdaki komutu çalıştırın:

         <Tabs>
           { /* npx */ }

           <TabItem label="npx">
             ```shellscript filename="Terminal"
             npx create-aptos-dapp@latest
             ```
           </TabItem>

           { /* pnpx */ }

           <TabItem label="pnpx">
             ```shellscript filename="Terminal"
             pnpx create-aptos-dapp@latest
             ```
           </TabItem>
         </Tabs>

      2. İstemleri takip edin

         Yeni dApp'iniz için bir isim, [şablon](/build/create-aptos-dapp#templates) ve ağ seçmek üzere CLI istemlerini takip edin.

         ![cad](~/images/cad-video.gif)

      3. Yeni dApp'inizi oluşturmaya ve özelleştirmeye başlayın!

         Yeni projenize gidin ve geliştirmeye devam etmek için tercih ettiğiniz IDE'de açın!

         Sonraki adımlar için oluşturulan `README.md` dosyasını takip edin.

      4. Okumaya devam edin

         <CardGrid>
           <LinkCard href="/build/create-aptos-dapp" title="Aptos DApp Oluşturun" description="Araç hakkında daha fazla bilgi edinin" />

           <LinkCard href="/build/create-aptos-dapp#templates" title="Şablonlar" description="Hazır şablonlara göz atın" />

           <LinkCard href="/build/create-aptos-dapp/faq" title="SSS" description="Yaygın sorunlar ve sorular için yardım alın" />
         </CardGrid>
    </Steps>
  </TabItem>
</Tabs>
