Geliştirme türüne bağlı olarak ortamınızı kurmanın kolay bir yolu burada.

<Tabs>
  {/* Frontend */}

  <TabItem label="Frontend">
    <Steps>
      1. Frontend Projesi Başlatma

         İşte popüler seçeneklerin bazı örnekleri:

         <Tabs>
           <TabItem label="Next.js">
             ```shellscript filename="Terminal"
             pnpx create-next-app@latest
             ```
           </TabItem>

           <TabItem label="Vite (TS)">
             ```shellscript filename="Terminal"
             pnpx create vite my-aptos-app --template react-ts
             ```
           </TabItem>
         </Tabs>

      2. @aptos-labs/ts-sdk Yükleme

         ```shellscript npm2yarn
         npm i @aptos-labs/ts-sdk
         ```

      3. TS SDK Kurulumu

         <CardGrid>
           <LinkCard href="/tr/build/sdks/ts-sdk/quickstart" title="TS SDK Hızlı Başlangıç" description="Hesabınızı kurma, ağ, musluk kullanma, işlem gönderme / simüle etme ve daha fazlasını nasıl yapacağınızı görün" />
         </CardGrid>

      4. Uygulamanızı oluşturun!

         Frontend'inizde Aptos kullanmak için geliştirici kurulumu artık tamamlandı.
         Geliştirme sürecini hızlandıran diğer araçlarımızı inceleyin

         <CardGrid>
           <LinkCard href="/tr/build/indexer" title="İndeksleyici" description="Bakiyeler, işlem aktivitesi, token verileri ve daha fazlası gibi zincir üstü durumu verimli bir şekilde sorgulayın" />

           <LinkCard href="https://github.com/aptos-labs/aptos-ts-sdk/tree/main/examples/typescript" title="TS SDK Örnekleri" description="TS SDK'yı nasıl kullanacağınıza dair 20+ örnek" target="_blank" />

           <LinkCard href="https://build.aptoslabs.com/" title="Aptos Build" description="Fullnode API / İndeksleyiciler için hız sınırlarına mı takıldınız? Buradan API Anahtarı alın" target="_blank" />
         </CardGrid>
    </Steps>
  </TabItem>

  {/* Smart Contract */}

  <TabItem label="Akıllı Sözleşme">
    <Steps>
      1. CLI Yükleme

         <CardGrid>
           <LinkCard href="/tr/build/cli" title="Aptos CLI" description="Aptos CLI'nin nasıl yükleneceğine dair talimatlar" />
         </CardGrid>

      2. Editör veya IDE Kurulumu

         Move geliştirmeyi kolaylaştırmak için tercih ettiğiniz editöre aşağıdaki uzantıları ekleyin

         <Tabs>
           <TabItem label="JetBrains IDEs">
             <CardGrid>
               <LinkCard href="https://plugins.jetbrains.com/plugin/14721-move-language" title="Move on Aptos" description="CLion, Rust Rover, WebStorm, IntelliJ gibi JetBrains IDE'leri için dil sunucusu ve sözdizimi vurgulayıcısı" target="_blank" />
             </CardGrid>
           </TabItem>

           <TabItem label="VSCode">
             <CardGrid>
               <LinkCard href="https://marketplace.visualstudio.com/items?itemName=AptosLabs.move-on-aptos" title="Move on Aptos uzantısı" description="VSCode için tam özellikli Aptos Move IDE'si" target="_blank" />
             </CardGrid>
           </TabItem>

           <TabItem label="Cursor, Kiro, vb.">
             <CardGrid>
               <LinkCard href="https://open-vsx.org/extension/aptoslabs/move-on-aptos" title="Move on Aptos uzantısı" description="OpenVSX destekli platformlar için tam özellikli Aptos Move IDE'si" target="_blank" />
             </CardGrid>
           </TabItem>
         </Tabs>

      3. Akıllı Sözleşme Oluşturma

         Uygulama klasörünüze gidin ve şu komutu kullanarak yeni bir akıllı sözleşme başlatın:

         ```shellscript filename="Terminal"
         aptos move init --name my_todo_list
         ```

      4. Akıllı Sözleşmeyi Oluşturun, Derleyin ve Dağıtın!

         Akıllı sözleşmeler için Aptos kullanmaya yönelik geliştirici kurulumu artık tamamlandı.
         Daha fazla bilgi için aşağıdaki Dapp öğreticisi bağlantısına bakın

         <CardGrid>
           <LinkCard href="/tr/build/guides/build-e2e-dapp/1-create-smart-contract#what-is-a-movetoml-file" title="Akıllı Sözleşme Oluşturma Kılavuzu" description="Move ile akıllı sözleşme kurulumu için kolay yapılacaklar listesi kılavuzu" />
         </CardGrid>
    </Steps>
  </TabItem>

  {/* create-aptos-dapp */}

  <TabItem label="Aptos Dapp Oluştur">
    <Steps>
      1. create-aptos-dapp Yükleme

         Saniyeler içinde şablondan bir dApp yüklemek için aşağıdaki komutu çalıştırın:

         <Tabs>
           { /* npx */ }

           <TabItem label="npx">
             ```shellscript filename="Terminal"
             npx create-aptos-dapp@latest
             ```
           </TabItem>

           { /* pnpx */ }

           <TabItem label="pnpx">
             ```shellscript filename="Terminal"
             pnpx create-aptos-dapp@latest
             ```
           </TabItem>
         </Tabs>

      2. Yönergeleri Takip Edin

         Yeni dApp'iniz için bir ad, [şablon](/tr/build/create-aptos-dapp#templates) ve ağ seçmek için CLI yönergelerini takip edin.

         ![cad](~/images/cad-video.gif)

      3. Yeni dApp'inizi oluşturmaya ve özelleştirmeye başlayın!

         Yeni projenize gidin ve oluşturmaya devam etmek için favori IDE'nizde açın!

         Sonraki adımlar için oluşturulan `README.md` dosyasını takip edin.

      4. Okumaya devam edin
