Farklılıklar ve benzerlikler hakkında daha fazla bilgi için [Aptos Learn](https://learn.aptoslabs.com/en/tutorials/solana-to-aptos-guide/cheat-sheet?workshop=solana-to-aptos)'e bakın

|                              | Solana                                                                                                             | Aptos                                                                 |
|------------------------------|--------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------|
| **Akıllı Sözleşmeler**       | Rust, SVM                                                                                                          | Move, MoveVM                                                          |
| **İşlem Ücretleri**          | Düşük                                                                                                              | Düşük                                                                 |
| **Paralelleştirme**          | Pesimist paralelleştirme, tüm yazma hesaplarını beyan etme gerekliliği                                           | Optimist paralelleştirme, zincir sizin için yazma hesaplarını çıkarır |
| **Sözleşme Hesabı Desteği**  | PDA Hesabı                                                                                                         | [Nesne](/build/smart-contracts/objects) veya [kaynak hesabı](/build/smart-contracts/resource-accounts)(bunun yerine nesne kullanılması önerilir) |
| **Veri Depolama**            | Veri programlara ait hesapta depolanır                                                                             | Veri kullanıcı hesabında veya nesnede kaynak olarak depolanır        |
| **Depolama Seviyesi**        | Program seviyesi                                                                                                   | Nesnede depolandığında küresel                                        |
| **Depolama Zihniyeti**       | Kullanıcı verileri hesapta dağıtılmış olarak depolanır                                                             | Kullanıcı verileri nesnede dağıtılmış olarak depolanır                |
| **Kod Örneği**               | [Solana'da yapılacaklar listesi sözleşmesi](https://github.com/aptos-labs/move-by-examples/tree/main/advanced-todo-list/solana) | [Aptos'ta yapılacaklar listesi sözleşmesi](https://github.com/aptos-labs/move-by-examples/tree/main/advanced-todo-list/aptos) |
| **Çağıran Kimliği**          | `signer`                                                                                                           | `signer`                                                              |
| **Yükseltilebilirlik**       | Program yükseltilebilir                                                                                           | Modül yükseltilebilir                                                 |
| **Dispatch Türü**            | Statik dispatch                                                                                                    | Statik dispatch                                                       |
| **FT Standardı**              | Token programı                                                                                                     | [Coin](/build/smart-contracts/aptos-coin) (legacy) ve [Fungible Asset Standardı](/build/smart-contracts/fungible-asset) |
| **NFT Standardları**         | Token programı                                                                                                     | [Digital Asset Standardı](/build/smart-contracts/digital-asset)       |
| **Blok Zinciri Etkileşimi**  | Solana web3.js kütüphanesi                                                                                         | [Aptos TypeScript SDK](/build/sdks/ts-sdk)                            |
