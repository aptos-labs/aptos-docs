İşlemler, Aptos blok zincirindeki verileri değiştirmenin temel yoludur. Onları paket göndermek gibi düşünün: ne gönderdiğinizi, kime gittiğini belirtmeniz ve teslimatın onaylanana kadar takip etmeniz gerekir. Blok zinciri terimlerinde, işlemler coin transfer etmenize, akıllı sözleşme fonksiyonlarını çağırmanıza ve zincir üstü durumu güncellemenize olanak tanır.

Bu öğretici, Aptos blok zincirinde ilk işleminizi oluşturma ve gönderme konusunda size rehberlik edecektir. Şunları öğreneceksiniz:

1. Geliştirme ortamınızı kurma
2. Test hesapları oluşturma ve fonlama
3. Coin transfer etmek için işlem oluşturma
4. Maliyetleri tahmin etmek için işlemi simüle etme
5. İşlemi imzalama ve gönderme
6. İşlemin başarıyla yürütüldüğünü doğrulama

<Aside type="note">
  Bu öğretici Aptos blok zinciri kavramları üzerine kuruludur. Blok zinciri geliştirmesinde yeniyseniz, endişelenmeyin - yol boyunca temel kavramları açıklayacağız.

  Tam kod örneğine [buradan](#full-code-sample) atlayabilir veya adım adım anlatım için okumaya devam edebilirsiniz.
</Aside>

## 2. Test Hesapları Oluşturma

<Tabs>
  <TabItem label="TypeScript">
    Blok zincirinde, tüm işlemler bir hesaptan gelmelidir. İki test hesabı oluşturalım: coin göndermek için bir tane (Alice) ve almak için bir tane (Bob).

    <Steps>
      1. İstemciyi kurma

         İlk olarak, blok zincirine bağlanacak Aptos istemcisini başlatmamız gerekir. `transaction.ts` dosyasını editörünüzde açın ve ekleyin:

         ```typescript filename="transaction.ts"
         import {
           Account,
           Aptos,
           AptosConfig,
           Network,
         } from "@aptos-labs/ts-sdk";

         async function main() {
           // Aptos istemcisini başlat
           const config = new AptosConfig({ network: Network.DEVNET });
           const aptos = new Aptos(config);

           console.log("Aptos devnet'e bağlandı");
         }

         main();
         ```

         <Aside type="note">
           Bu kod devnet'e bağlanır - Aptos'un test ağı. Bu, gerçek para harcamadan deneyim yapmanıza olanak tanır.
         </Aside>

      2. Hesapları oluşturma

         Şimdi Alice ve Bob hesaplarını ekleyin:

         ```typescript filename="transaction.ts"
         // Alice ve Bob hesaplarını oluştur
         const alice = Account.generate();
         const bob = Account.generate();

         console.log("Alice adresi:", alice.accountAddress.toString());
         console.log("Bob adresi:", bob.accountAddress.toString());
         ```

         <Aside type="note">
           Bir hesap oluşturmak, özel anahtar/genel anahtar çifti oluşturmak anlamına gelir. Özel anahtar, işlemleri imzalamak için kullanılır, genel anahtar (ve türetilen adres) ise hesabı tanımlamak için kullanılır.
         </Aside>

      3. Hesapları fonlama

         Test için Alice hesabını devnet faucet'den fonlayalım:

         ```typescript filename="transaction.ts"
         // Alice'i faucet ile fonla
         console.log("Alice'i fonlama...");
         await aptos.fundAccount({
           accountAddress: alice.accountAddress,
           amount: 100_000_000, // 1 APT (APT'nin 8 ondalık basamağı vardır)
         });

         console.log("Alice fonlandı!");
         ```

         <Aside type="note">
           Faucet, test ağlarında ücretsiz test token'ları sağlayan bir hizmettir. Gerçek APT değil, yalnızca test için kullanılır.
         </Aside>
    </Steps>
  </TabItem>

## 3. İşlem Oluşturma

<Tabs>
  <TabItem label="TypeScript">
    Artık fonlanmış hesaplarımız olduğuna göre, Alice'den Bob'a coin transfer etmek için bir işlem oluşturalım. Bu, ne göndermek istediğinizi ve kime göndereceğinizi belirten bir form doldurmak gibidir.

    <Steps>
      1. İşlem yapısını anlama

         Aptos'ta bir işlem birkaç temel bileşene sahiptir:

         1. **Gönderen**: İşlemi başlatan hesap (Alice)
         2. **Fonksiyon**: Çağrılacak zincir üstü fonksiyon (bu durumda coin transfer)
         3. **Argümanlar**: Fonksiyon tarafından ihtiyaç duyulan veri (alıcı adresi ve miktar)
         4. **Gas parametreleri**: Maksimum gas miktarı ve gas birim fiyatı
         5. **Son kullanma zamanı**: İşlemin yürütülmezse artık geçerli olmayacağı zaman
         6. **Sıra numarası**: Tekrar saldırılarını önleyen bir sayaç

         <Aside type="note">
           Aptos işlemlerindeki tüm veriler, blok zinciri kullanımı için tasarlanmış kompakt ve deterministik bir format olan Binary Canonical Serialization (BCS) kullanılarak serileştirilir. SDK bu serileştirmeyi sizin için halleder.

           BCS, işlem verilerinin farklı platformlar ve diller arasında tutarlı bir şekilde kodlanmasını sağlar, bu da aynı işlemin farklı implementasyonlar çalıştıran düğümler tarafından işlenebileceği bir blok zinciri için kritiktir.
         </Aside>

      2. İşlemi oluşturma

         Alice'den Bob'a 1000 octa transfer eden bir işlem oluşturmak için kod ekleyelim:

         ```typescript filename="transaction.ts"
         // Alice'den Bob'a transfer işlemi oluştur
         const transaction = await aptos.transaction.build.simple({
           sender: alice.accountAddress,
           data: {
             function: "0x1::aptos_account::transfer",
             functionArguments: [bob.accountAddress, 1000],
           },
         });

         console.log("İşlem oluşturuldu");
         ```

         <Aside type="note">
           "0x1::aptos\_account::transfer" standart bir Aptos fonksiyonudur. "0x1" core framework'ün adresidir, "aptos\_account" modül adıdır ve "transfer" fonksiyon adıdır.
         </Aside>
    </Steps>
  </TabItem>

  <TabItem label="Python">

## 4. İşlemi Simüle Etme

<Tabs>
  <TabItem label="TypeScript">
    ```typescript filename="transaction.ts"
    // İşlemi simüle et
    const [simulateResult] = await aptos.transaction.simulate.simple({
      signerPublicKey: alice.publicKey,
      transaction,
    });

    console.log("Simülasyon sonucu:", simulateResult);
    console.log("Gas kullanımı:", simulateResult.gas_used);
    ```
  </TabItem>

  <TabItem label="Python">

    print(f"Gas kullanımı: {simulated_result[0]['gas_used']}")

## Tam Kod Örneği

Bu öğreticide ele aldığımız tüm kod parçacıklarını birleştiren tam kod örnekleri aşağıda:

<Tabs>
  <TabItem label="TypeScript">
    ```typescript filename="transaction.ts"
    import {
      Account,
      Aptos,
      AptosConfig,
      Network,
    } from "@aptos-labs/ts-sdk";

    async function main() {
      // Aptos istemcisini başlat
      const config = new AptosConfig({ network: Network.DEVNET });
      const aptos = new Aptos(config);

      console.log("Aptos devnet'e bağlandı");

      // Alice ve Bob hesaplarını oluştur
      const alice = Account.generate();
      const bob = Account.generate();

      console.log("Alice adresi:", alice.accountAddress.toString());
      console.log("Bob adresi:", bob.accountAddress.toString());

      // Alice'i faucet ile fonla
      console.log("Alice'i fonlama...");
      await aptos.fundAccount({
        accountAddress: alice.accountAddress,
        amount: 100_000_000, // 1 APT
      });

      console.log("Alice fonlandı!");

      // Alice'den Bob'a transfer işlemi oluştur
      const transaction = await aptos.transaction.build.simple({
        sender: alice.accountAddress,
        data: {
          function: "0x1::aptos_account::transfer",
          functionArguments: [bob.accountAddress, 1000],
        },
      });

      console.log("İşlem oluşturuldu");

      // İşlemi simüle et
      const [simulateResult] = await aptos.transaction.simulate.simple({
        signerPublicKey: alice.publicKey,
        transaction,
      });

      console.log("Gas kullanımı:", simulateResult.gas_used);

      // İşlemi imzala ve gönder
      const committedTransaction = await aptos.transaction.signAndSubmitTransaction({
        signer: alice,
        transaction,
      });

      console.log("İşlem gönderildi:", committedTransaction.hash);

      // İşlemin tamamlanmasını bekle
      await aptos.waitForTransaction({ transactionHash: committedTransaction.hash });
      
      console.log("İşlem başarıyla tamamlandı!");
      
      // Bakiyeleri kontrol et
      const aliceBalance = await aptos.getAccountAPTAmount({
        accountAddress: alice.accountAddress,
      });
      const bobBalance = await aptos.getAccountAPTAmount({
        accountAddress: bob.accountAddress,
      });
      
      console.log("Alice bakiyesi:", aliceBalance);
      console.log("Bob bakiyesi:", bobBalance);
    }

    main();
    ```
  </TabItem>

  <TabItem label="Python">
    ```python filename="transaction.py"
    import asyncio
    from aptos_sdk.async_client import RestClient
    from aptos_sdk.account import Account
    from aptos_sdk.bcs import Serializer
    from aptos_sdk.transactions import EntryFunction, TransactionArgument

    async def main():
        # Aptos istemcisini başlat
        client = RestClient("https://fullnode.devnet.aptoslabs.com/v1")
        
        print("Aptos devnet'e bağlandı")

        # Alice ve Bob hesaplarını oluştur
        alice = Account.generate()
        bob = Account.generate()

        print(f"Alice adresi: {alice.address()}")
        print(f"Bob adresi: {bob.address()}")

        # Alice'i faucet ile fonla
        print("Alice'i fonlama...")
        await client.fund_account(alice.address(), 100_000_000)  # 1 APT
        
        print("Alice fonlandı!")

        # Alice'den Bob'a transfer işlemi oluştur
        payload = EntryFunction.natural(
            "0x1::aptos_account",
            "transfer",
            [],
            [TransactionArgument(bob.address(), Serializer.struct),
             TransactionArgument(1000, Serializer.u64)]
        )

        print("İşlem oluşturuldu")

        # İşlemi simüle et
        simulated_result = await client.simulate_transaction(alice, payload)
        
        print(f"Gas kullanımı: {simulated_result[0]['gas_used']}")

        # İşlemi imzala ve gönder
        signed_transaction = alice.sign_transaction(payload)
        submitted_transaction = await client.submit_transaction(signed_transaction)
        
        print(f"İşlem gönderildi: {submitted_transaction}")

        # İşlemin tamamlanmasını bekle
        await client.wait_for_transaction(submitted_transaction)
        
        print("İşlem başarıyla tamamlandı!")
        
        # Bakiyeleri kontrol et
        alice_balance = await client.account_balance(alice.address())
        bob_balance = await client.account_balance(bob.address())
        
        print(f"Alice bakiyesi: {alice_balance}")
        print(f"Bob bakiyesi: {bob_balance}")

    if __name__ == "__main__":
        asyncio.run(main())
    ```
  </TabItem>
</Tabs>
