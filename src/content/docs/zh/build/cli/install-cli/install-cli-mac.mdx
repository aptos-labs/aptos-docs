---
title: "在 Mac 上安装 Aptos CLI"
---

对于 Mac,安装 Aptos CLI 的最简单方法是使用包管理器 `brew`.



import { Aside, Steps } from '@astrojs/starlight/components';



# 安装



<Steps>

1. 确保您安装了 brew https://brew.sh/

2. 打开一个新的终端并输入以下命令.

   ```shellscript filename="Terminal"
   brew update
   brew install aptos
   ```

3. 打开另一个终端并运行 aptos help 来验证 CLI 是否安装.

   ```shellscript filename="Terminal"
   aptos help
   ```

</Steps>



<Aside type="caution">
  如果 `brew` 对您不起作用,您可以尝试这里提供的步骤:[安装特定版本的 Aptos CLI(高级)](/zh/build/cli/install-cli/install-cli-specific-version)
</Aside>

# 升级 CLI

使用 brew 升级 CLI 只需 2 个命令:

```shellscript filename="Terminal"
brew update
brew upgrade aptos
```
