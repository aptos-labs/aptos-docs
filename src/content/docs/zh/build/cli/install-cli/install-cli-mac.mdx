在 Mac 上,安装 Aptos CLI 最简单的方法是使用包管理器 `brew`.

# 安装

<Steps>
  1. 确保已安装 brew https://brew.sh/ .

  2. 打开新的终端并输入以下命令.

     ```shellscript filename="Terminal"
     brew update
     brew install aptos
     ```

  3. 打开另一个终端并运行 aptos help 来验证 CLI 是否已安装.

     ```shellscript filename="Terminal"
     aptos help
     ```
</Steps>

<Aside type="caution">
  如果 `brew` 不适合你,你可以尝试这里的步骤:[安装特定版本的 Aptos CLI (高级)](/zh/build/cli/install-cli/install-cli-specific-version)
</Aside>

# 升级 CLI

使用 brew 升级 CLI 只需要 2 个命令:

```shellscript filename="Terminal"
brew update
brew upgrade aptos
```
