---
title: "通过 Aptos CLI 管理网络节点"
---







如果您正在运行[验证者节点或验证者全节点(VFN)](/zh/network/nodes/validator-node),您可以使用命令行界面(CLI)与您的节点交互.

具体来说,您可以使用 CLI 执行以下操作:

1. [管理您拥有的质押池](/zh/network/nodes/validator-node/connect-nodes/staking-pool-operations).
2. [对提案进行投票](/zh/network/nodes/validator-node/connect-nodes/staking-pool-voter).

除此之外,您可以运行以下帮助命令,查看更多与您的节点相关的执行特定的操作命令:

```shellscript filename="Terminal"
aptos node --help
```
