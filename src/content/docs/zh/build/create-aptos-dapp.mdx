`create-aptos-dapp` 为 dapp 开发者构建模板项目，以便在 Aptos 网络上轻松创建前端和智能合约。

## 为什么使用 create-aptos-dapp？

- **模板化设置**：`create-aptos-dapp` 为您生成预定义的端到端 dapp 模板和配置文件。它节省了手动设置项目结构的时间，而手动设置既耗时又容易出错。
- **合约目录**：`create-aptos-dapp` 生成包含 Move 智能合约模块基本结构的 `contract` 目录。
- **最佳实践**：`create-aptos-dapp` 融入了为 Aptos 网络开发的最佳实践和结构建议。
- **内置 Move 命令**：`create-aptos-dapp` 包含用于常见任务的内置命令，如初始化 Move 编译器、编译和在链上发布智能合约。

## 前提条件

- [node 和 npm](https://nodejs.org/en) (npm ≥ 5.2.0)
- [Python 3.6+](https://www.python.org/)

## 模板

`create-aptos-dapp` 为您提供预制的端到端 dapp 模板，即具有配置和美观 UI 的现成 dapp，让您开始在 Aptos 上创建 dapp。

模板的目标是：

1. 通过端到端 dapp 模板示例让用户熟悉不同的 Aptos 标准。
2. 教育用户如何在 Aptos 上构建 dapp，从前端层到智能合约层以及中间的所有内容。
3. 为用户提供预制模板以快速部署简单的 dapp

- [样板模板](https://learn.aptoslabs.com/en/dapp-templates/boilerplate-template)
- [NFT 铸造 dapp 模板](https://learn.aptoslabs.com/en/dapp-templates/nft-minting-template)
- [代币铸造 dapp 模板](https://learn.aptoslabs.com/en/dapp-templates/token-minting-template)

- [自定义索引器模板](https://learn.aptoslabs.com/en/dapp-templates/custom-indexer-template)
