Dutch auction 示例是本指南更接近真实应用的一半。它不只讲语法，还会展示 Aptos 应用如何围绕 Object 和框架标准组织起来。

## 这个示例会覆盖什么

- 如何 mint 与配置 Digital Asset
- 如何用 Object 建模拍卖状态
- 如何用 Fungible Asset 接收付款
- 如何发出事件并暴露 view function
- 如何在没有 Solidity modifier 的情况下实现 only-owner 行为
