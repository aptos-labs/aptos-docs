## Solidity

```solidity
event AuctionCreated(uint256 nftId);
```

## Move

```move
#[event]
struct AuctionCreated has drop, store {
    auction: Object<Auction>,
}
```

这里最有意思的差异是：Move 事件可以直接携带强类型的 `Object<Auction>`，这样下游读取方拿到的不是单纯的数字 ID，而是能直接定位到拍卖容器的对象句柄。

和其他 Aptos 事件一样，下游通常会通过 [Indexer](/zh/build/indexer) 或交易 API 去读取这些事件。
