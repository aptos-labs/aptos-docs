<Aside type="caution" title="旧版钱包 API 已被移除">
  通过浏览器全局变量（如 `window.aptos` 或 `window.petra`）直接访问钱包的方式已被**移除**。这些旧版 API 在现代钱包版本中已不再可用（例如，Petra 已完全移除了这些 API）。所有 dapp 必须使用钱包适配器配合 [AIP-62](https://github.com/aptos-foundation/AIPs/blob/main/aips/aip-62.md) 标准进行钱包集成。
</Aside>

Aptos 生态系统中有两种钱包适配器标准：

1. [Aptos 钱包适配器](#aptos-钱包适配器) 由 Aptos Labs 提供
2. [OKX Connect](#okx-connect) 由 OKX 提供

## Aptos 钱包适配器

Aptos Labs 提供的 Aptos 钱包适配器为 Aptos dapp 和 Aptos 钱包之间的通信提供统一接口。

对于 dapp 开发者，这意味着你可以连接到任何已集成钱包适配器的 Aptos 钱包，无需为每个钱包编写自定义代码。这在 [AIP-62](https://github.com/aptos-foundation/AIPs/blob/main/aips/aip-62.md) 中有详细描述，该提案定义了现代钱包标准和钱包自动检测机制。

对于 Aptos 钱包提供商，集成 AIP-62 意味着你的钱包将与任何使用钱包适配器的 dapp 兼容。

### 面向 Aptos Dapp 开发者

参阅 [Dapp 开发者钱包适配器指南](/zh/build/sdks/wallet-adapter/dapp)，了解如何使用钱包适配器（通过 [Wallet Adapter React 包](https://github.com/aptos-labs/aptos-wallet-adapter/tree/main/packages/wallet-adapter-react)）。

### 面向 Aptos 钱包提供商

参阅以下指南之一，了解如何实现 dapp 可以连接的钱包适配器插件：

1. 面向[浏览器扩展钱包](/zh/build/sdks/wallet-adapter/browser-extension-wallets)（例如 [Petra](https://chromewebstore.google.com/detail/petra-aptos-wallet/ejjladinnckdgjemekebdpeokbikhfci?hl=en)）
2. 面向 [SDK 钱包](/zh/build/sdks/wallet-adapter/wallets)（例如 [AptosConnect](https://aptosconnect.app)）

## OKX Connect

OKX Connect 适配器为 Aptos dapp 提供连接 OKX 钱包和其他支持 OKX Connect 标准的钱包的接口。你可以在 [OKX Connect 文档](https://web3.okx.com/build/dev-docs/sdks/app-connect-aptos)中找到更多关于 Aptos dapp 使用 OKX Connect 的信息。
