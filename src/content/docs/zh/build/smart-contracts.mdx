Aptos 合约使用 Move 语言编写,这是一种用于安全,沙盒化和形式化验证编程的下一代语言,被多个区块链采用.
Move 允许开发者编写灵活管理和转移资产的程序,同时为这些资产提供安全性和攻击防护.
