<CardGrid>
  <ListCard title="功能" iconName="lightning">
    - [无钥匙账户](/zh/build/guides/aptos-keyless) 使用社交登录，无需钱包或助记词即可注册和签名。
    - [新！通行密钥 (Passkeys)](https://github.com/aptos-foundation/AIPs/blob/main/aips/aip-66.md) 无缝注册与恢复账户，无需私钥或助记词；在 Apple 和 Android 设备间自动同步。
    - [赞助交易](/zh/build/guides/sponsored-transactions) 为用户支付 Gas 费，让他们在无 APT 的情况下使用 dApp。
    - [新！无序交易 (Orderless Transactions)](/zh/build/guides/orderless-transactions) 可无序发送交易并具备防重放 nonce；适用于高并发应用。
    - [链上随机性](/zh/build/smart-contracts/randomness) 提供可验证的随机数，用于公平游戏、抽奖和空投。
    - [并行执行与约 250 ms 出块](/zh/network/blockchain/execution) 通过 Block-STM（软件事务内存）实现安全确定性并行执行，高吞吐且出块迅速。
  </ListCard>


  <ListCard title="工具" iconName="terminal">
    - [新！AI 开发 (MCP)](/zh/build/ai/aptos-mcp) 将 Cursor、Claude Code 与 Aptos NPM MCP 结合，快速构建现代 Move 全栈模板，并直接通过 Geomi 完成基础设施设置（项目、API 密钥、Gas 赞助等）。
    - [Aptos CLI](/zh/build/cli) 初始化、编译、测试并发布 Move；管理账户和密钥；运行本地区块链网络。
    - [官方 SDK](/zh/build/sdks) 支持 TypeScript、Go、Java（即将推出）、Python、Rust、.NET。
    - [索引器 API](/zh/build/indexer/indexer-api) 查询链上数据用于应用与分析。
    - [新！AskAptos AI 聊天机器人](#) 基于 Aptos 文档（RAG）的 AI 助手：探索功能、查询文档、规划或优化 PRD；立即在右上角体验！
    - [测试网水龙头](/zh/network/faucet) 需要测试网 APT？使用 Google 登录为测试账户充值；可通过钱包或 CLI 创建账户。
  </ListCard>
